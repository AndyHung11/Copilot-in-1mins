# client-monthly-report

## 技能用途
將關係經理的客戶接觸紀錄、持倉變化與再平衡討論，整理成客戶可閱讀的財富管理月報。輸出需專業、清楚、合規安全，並保留必要數字來源。

## 輸出模式判斷
- 若使用者說「email 版」、「email version」、「Outlook email」、「寄給客戶的版本」：輸出 responsive、inline-styled 的 HTML email，可直接貼到 Outlook。
- 其他情況：輸出 standalone visual HTML report，包含清楚標題、卡片區塊、表格與簡單視覺層級。

## 固定章節
1. **本月摘要**：3–5 點，聚焦客戶目標、資產變化與需注意事項。
2. **資產配置與變化**：列出本月買進、賣出、到期、現金比重與配置差異。
3. **績效回顧**：只使用輸入資料提供的期間報酬或估值變化；若缺數字，標註「原始資料未提供」。
4. **市場觀點**：以中性語氣連結本月市場背景與客戶配置，不誇大。
5. **下月建議**：列出可討論的再平衡、風險控管或追蹤事項；不得寫成確定指令。
6. **合規聲明**：必須包含投資風險與非保證報酬聲明。

## 守則與防護欄
- 不得使用「保證收益」、「穩賺」、「無風險」、「一定上漲」等語句。
- 不得杜撰持倉、金額、績效或客戶偏好；缺漏處請列為待確認。
- 帳號、身分證字號、完整生日與電話必須遮罩，例如 ***-***-7890。
- 過濾內部註記、銷售目標、費率抱怨、同仁私下評論與不適合客戶閱讀的內容。
- 所有建議需以「可考慮」、「建議討論」、「需依風險屬性確認」表述。
- 結尾加入：本報告僅供資訊參考，非投資建議或招攬；投資涉及風險，過去績效不代表未來結果。
