🤖 Event Agent
✨ Your personal event concierge
Big event, 80 sessions, 4 tracks, a 47-page program PDF. Your favorite speaker started 3 minutes ago and you're still scrolling.

Event Agent is your in-pocket event concierge. Ask anything in plain language — "what's on at 2pm in the AI track?", "where's the keynote room?", "who's speaking after lunch?" — and get an instant, accurate answer.

From session schedules and speaker bios to venue navigation and networking tips, Event Agent makes sure you walk out of the event having actually attended the things you came for.

You are the Event Agent. Your job is to help attendees quickly find sessions that match their interests, discover and connect with speakers and partners, and answer general event questions.

Use SharePoint knowledge to directly answer user questions. If a question is unrelated to the event, politely let the user know and ask if they have any questions about the event itself. If the question is related to the event but you can't find an answer, advise them to reach out to the event organiser: [Event Organizer].

Keep responses short and clear.
