THIS ISSUE · PROMPT
📝 Update a Word Doc with Tracked Changes
✨ Pull from meetings, emails & files to update the doc—redlined so you see every change 📝
You've got a Word document to update—a proposal, a status report, a template—and you need to fold in the latest meeting outcomes, email threads, or file data. Copying and pasting by hand is slow and a fast way to wreck the formatting, numbering, and styles you carefully set up.

This prompt has Copilot, right inside Word, update, adapt, or fill in your open document using relevant information from other documents or sources—while keeping the existing formatting and structure intact. The key part: every update comes in as a tracked change (redline), so you review each adjustment and accept or reject on your terms.

How to use it: Open the doc you want to update in Word, bring up Copilot, paste the prompt below (swap the brackets for your sources), and it applies the updates as redlines.

No more "will the AI mangle my formatting?"—you see every change and stay in control ✨.
  📋 Prompt Template (English)
📝 Update a Word doc with tracked changes (bring up Copilot inside Word and paste)

Update the open Word document using the relevant information from [meetings, emails, files, notes]. Preserve the document's existing formatting and structure, and track all edits as redlined changes so I can review them before accepting.
  📋 提示詞範本（繁體中文版）
📝 用追蹤修訂更新 Word 文件（在 Word 內叫出 Copilot 貼上）

請使用來自 〔會議、電子郵件、檔案、筆記〕 的相關資訊，更新目前開啟的 Word 文件。保留文件既有的格式與結構，並將所有修改以追蹤修訂（紅線）方式標記，讓我可以逐一檢視後再決定接受。
💡 Usage Tips
1. 🔓 Turn on WorkIQ first — to pull from your meetings, emails, and files, enable WorkIQ (formerly Work mode) in Copilot, or it can't reach your org content.
2. 📂 Name specific sources — replace the brackets with concrete sources (e.g. "last Thursday's project meeting notes," "the quote email to the client") so Copilot knows where to look and fills accurately.
3. 👀 Review one by one — after it runs, open Word's Review tab to walk through each tracked change, then Accept All once verified and reject any you're unsure about.
