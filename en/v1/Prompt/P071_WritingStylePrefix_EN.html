✍️ Writing Style Prefix
✨ Make Copilot write in your voice 🎯
Default Copilot output is correct, smooth, and… completely generic. It doesn't sound like you—and rewriting every draft defeats the point of using AI.

This prompt walks you through building a personal style prefix: Copilot studies how you actually write, captures your phrasing, rhythm, and quirks, and turns it into a reusable instruction block you can paste at the top of any prompt.

From now on, output sounds like you wrote it—because in tone and voice, you did.
  📋 Prompt Template (English)
Analyze my recent Teams messages and Outlook emails from the last 90 days. Based on this, identify and summarize my personal writing style covering: - Tone (formal vs. casual, direct vs. diplomatic) - Typical message length and structure - Recurring phrases or expressions I use - How I open and close messages - Any other patterns that make my writing recognizable. Then generate a short 'style prefix' of 3-6 sentences that I can paste in front of any future prompt. This prefix should instruct Copilot to match my personal tone, vocabulary, and style whenever it generates text for me, so the output sounds like it was written by me, not by an AI. Make the prefix ready to use immediately: written in first person, as if I'm giving Copilot a standing instruction.
  📋 提示詞範本（繁體中文版）
請分析我過去 90 天的 Teams 訊息與 Outlook 郵件。根據分析，識別並摘要我的個人寫作風格，包括：- 語氣（正式 vs 輕鬆，直接 vs 圓融）- 典型訊息長度與結構 - 我常用的詞語或表達方式 - 我如何開場與結尾 - 其他讓我的文字有辨識度的模式。接著，生成一段 3-6 句的「Style Prefix」，讓我可以貼在任何未來的 Prompt 前面。這個 Prefix 應該指示 Copilot，在為我生成文字時，要符合我的個人語氣、詞彙與風格——讓輸出看起來像是我寫的，不像 AI 生成的。請讓 Prefix 立即可用：以第一人稱撰寫，就像我在給 Copilot 一個永久的指令。
