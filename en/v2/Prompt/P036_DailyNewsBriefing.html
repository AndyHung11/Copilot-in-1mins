🚀 Copy This Prompt (English Version)
Open Copilot Chat (Web or Work) and paste the prompt below.
Create a daily briefing on news from [industry] related to [topic]. Structure each summary into a well-organized paragraph, beginning with a clear and assertive topic sentence that highlights the key point of the article. Summarize the main insights concisely while maintaining clarity and coherence. At the end of each summary, include "Source:" followed by the name of the publication, linking directly to the original article.
