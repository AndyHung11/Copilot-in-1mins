Copilot Memory is a new feature that remembers your preferences, work style, and key details — making every conversation more efficient and context-aware. This Prompt is your activation key: it asks you three simple questions and saves all your answers to your Memory profile in one go. Set it once, never repeat yourself again 🌿
🎁 What will you get?
💼 Work Style Memory
How you structure your day, whether you prefer summaries or detail, your meeting rhythm — Copilot remembers it all.
✍️ Writing Style Memory
Preferred tone, length, and greeting style — so every draft requires fewer edits.
🎯 Role Memory
Your title, team, and customer types — so Copilot answers from the perspective most useful to you.
🔧 Edit Anytime
Open Copilot Settings ➜ Memory to view, delete, or add any memory entry at any time.
🚀 Copy This Prompt (English Version)
Open Copilot Chat (Web or Work) and paste the prompt below.
Help me with getting started on my Copilot Memory. Ask me three simple questions about my work style, writing style and key facts about my role. Add the results to my Copilot Memory profile.
📋 提示詞範本（繁體中文版）
幫我啟動我的 Copilot Memory。請問我三個簡單的問題：分別關於我的 工作風格、寫作風格，以及 我的職務角色 的關鍵事實。請把我的回答加到我的 Copilot Memory 個人檔案中。
💡 Three Ways to Use It
1
🔍 Verify right after the first run
Ask Copilot: "Please repeat every Memory entry you just saved for me" — confirm it all went through.
2
➕ Add new memories anytime
In any conversation, say "Copilot, I prefer…" or "Please remember…" — watch for the small notification confirming it's saved.
3
🧹 Quarterly clean-up
Open Copilot Settings ➜ Memory, remove outdated preferences to keep your profile accurate.
