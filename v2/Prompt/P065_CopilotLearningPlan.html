Copilot 的功能很多，但大多數人面對一大堆教學資源，往往不知道該從哪裡下手，最後就是一直說「等我有空再學」——然後永遠沒有空。

這個 Prompt 讓 Copilot 直接幫你設計一份個人化的學習計畫——列出每天可以做的具體小任務，逐步把 Copilot 融入你的日常工作流程，同時每週推薦幫你快速入門的學習資源。

不需要一次學完所有功能，每天進步一點點，就是最好的導入策略！
🎁 你會得到什麼？
📅 每日可執行的 Copilot 任務
具體的每日小任務，讓 Copilot 自然融入工作流程
📚 每週推薦學習資源
精選的教學資源，不用自己大海撈針
📈 逐步養成習慣的計畫
循序漸進，從試用到習慣，真正讓 Copilot 發揮價值
🚀 直接複製這段 Prompt（繁中版）
打開 Copilot Chat - Work，把下面整段貼上去就能跑。
請建立一份訓練計畫，幫助我導入 M365 Copilot。包含我每天可以進行的日常活動清單，讓 Copilot 融入我的工作流程。每週加入實用資源，幫助我快速上手。
💬 English Prompt｜英文版
Create a training plan to help me adopt M365 Copilot. Include a list of daily activities I can do to build Copilot into my routine. Add helpful resources to get me started each week.
💡 三個情境式玩法
1
🎯 指定你的職務
加上「我是 [職稱]，主要工作是 [描述]」，讓計畫更貼近你的實際場景。
2
📌 從第一週任務開始
不用一次看完整份計畫，先做第一週的任務，體驗成就感再繼續。
3
👥 拉同事一起學
把這個 Prompt 分享給你的團隊，一起建立 Copilot 社群，互相分享心得。
