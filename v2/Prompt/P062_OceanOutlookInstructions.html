有沒有試過讓 Copilot 幫你起草郵件，結果讀起來怎麼都不像你說話的樣子？

這個 Prompt 讓 Copilot 先分析你過去 12 個月的職場溝通，從中推導出你的 OCEAN 特質（開放性、盡責性、外向性、親和性、情緒穩定性）並附上百分比。接著，它會把這些特質轉化成一份Markdown 格式的起草指示詞，你只要貼進「設定 → Copilot → 起草指示」，往後 Copilot 幫你起草的每一封郵件都會有你的風格。

一次設定，永久受益！
🎁 你會得到什麼？
🧬 OCEAN 人格特質分析
從過去 12 個月溝通記錄推導五大人格百分比
📝 現成 Markdown 起草指示詞
直接複製貼進 Copilot 設定 → 起草指示，立即生效
💬 範例措辭包含
符合你溝通風格的例句，讓語氣一次到位
🚀 直接複製這段 Prompt（繁中版）
打開 Copilot Chat - Work，把下面整段貼上去就能跑。
請建立一份基於 Markdown 的 Copilot Outlook 指示詞，反映我偏好的溝通方式——依據 Big Five/OCEAN 模型的溝通風格分析。需求：- 從我過去 12 個月的職場溝通（郵件和 Teams 訊息）中推導出 OCEAN 特徵值（開放性 Openness、盡責性 Conscientiousness、外向性 Extraversion、親和性 Agreeableness、情緒穩定性 Emotional Stability），附上百分比。- 用這些特徵引導語氣、語言、態度、結構與風格。- 指示詞應適合職場郵件，特別是與客戶和同事的溝通。- 使用清晰、主動的語言，以 Markdown 整潔結構化。- 納入符合我溝通風格的範例措辭。- 確保語言邏輯性（例如：依原訊息語言決定回覆語言）。
💬 English Prompt｜英文版
Please create a Markdown-based Copilot instruction for Outlook that reflects my preferred communication style – based on my communication profile according to the Big Five/OCEAN model. Requirements: - The OCEAN traits (Openness, Conscientiousness, Extraversion, Agreeableness, Emotional Stability) should be derived from my professional communication over the past 12 months (emails and Teams messages). - Integrate the OCEAN traits with percentage values. - Use these traits to guide the tone, language, attitude, structure and style. - The instruction should be suitable for professional emails – especially for communication with customers and colleagues. - Use clear, active language and structure the instruction cleanly in Markdown. - Include example phrases that match my communication style. - Ensure logical language handling (e.g. reply language based on original message).
💡 三個情境式玩法
1
⚙️ 貼進起草指示設定
Copilot 產出後，打開 Outlook → 設定 → Copilot → 起草指示，把整段 Markdown 貼上儲存。
2
🔁 每半年更新一次
你的溝通風格會隨時間改變，定期重跑讓指示詞保持貼近現在的你。
3
📋 測試效果
設定完後，請 Copilot 幫你起草一封郵件，看看語氣是否符合預期再微調。
