很容易搞混
需要把整份英文簡報翻成中文，或把中文版翻給外國客戶看？手動一張張翻譯是上個世紀的做法。這個 Prompt 讓 Copilot 自動建立一份全新翻譯版簡報，所有投影片上的文字都翻譯成你指定的語言，原始版本完整保留，翻譯版另存為新文件！一個指令，打開國際市場 🌏
🎁 你會得到什麼？
🌐 完整翻譯新簡報
自動建立全新翻譯版，所有投影片文字翻譯完整，格式與原版一致。
📋 原版完整保留
原始簡報不受影響，新版本另外儲存，方便並行管理。
🗣️ 任意目標語言
支援翻譯成任何你指定的語言，輕鬆應對不同市場需求。
⚡ 大幅節省時間
原本需要數小時的工作，幾秒鐘內完成，讓你專注在內容品質上。
🚀 直接複製這段 Prompt（繁中版）
打開 PowerPoint，把下面整段貼上去就能跑。
請建立這份簡報的新版本，並將所有投影片翻譯成 [選擇語言]。
💬 English Prompt｜英文版
Please create a new version of this slide deck and translate all the slides to [Choose Language]
💡 三個情境式玩法
1
🌏 把中文簡報翻成英文
要向總部報告，把原本給台灣客戶的中文簡報快速翻成英文版，省下大量時間。
2
🇯🇵 進軍東南亞市場本地化
要拓展日本市場，需要將英文產品簡報翻譯成日文版本，一個指令完成。
3
🔄 修改後同步翻譯版本
修改了英文版內容後，用這個 Prompt 快速產出更新後的中文版，保持同步。
