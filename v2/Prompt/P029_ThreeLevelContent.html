同一份報告，高層要看摘要，中階主管要看細節，技術專家要看深度分析 — 三份要分開重寫嗎？不用！這個 Prompt 讓 Copilot 把同一份內容自動分拆成三個版本：主管摘要、中階說明和深度分析，每個版本都針對目標受眾量身訂製。一份資料，三種版本，三組人都滿意 🎯
🎁 你會得到什麼？
👔 主管摘要版
高度濃縮，只留關鍵結論和決策依據，一眼掌握重點。
📊 中階說明版
補充背景和脈絡，讓主管和組長了解全貌而不迷失在細節中。
🔬 深度分析版
完整細節、數據和技術說明，讓專家和分析師得到所需深度。
⚡ 一次搞定三版
不需要重複作業，一個指令自動產出三個不同深度的版本。
🚀 直接複製這段 Prompt（繁中版）
打開 Word，把下面整段貼上去就能跑。
請將這份內容產出三個複雜度不同的版本：主管摘要版、中階說明版，以及深度分析版。
💬 English Prompt｜英文版
Create three versions of this content with different complexity levels: executive summary, mid-level explanation, and in-depth analysis.
💡 三個情境式玩法
1
👔 季報同時給不同層級看
同一份季度業績數據，自動產出給 C-level、主管和分析師的三個版本，各取所需。
2
📄 技術規格多版本說明
技術規格太難懂，用這個 Prompt 給業務和客戶各一個深度適中的說明版本。
3
🎯 提案資料客製化
董事會要看戰略重點，PM 要看執行細節，工程師要看技術規格 — 一份提案三種讀法。
