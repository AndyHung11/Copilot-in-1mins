OneNote 的筆記越記越多，卻找不到自己還有哪些事沒做完？這個 Prompt 讓 Copilot 掃描整個 OneNote 章節，把所有未完成的行動項目彙整成一張清楚的表格，包含說明和狀態。一秒掌握所有待辦，從此告別遺漏的焦慮 ✅
🎁 你會得到什麼？
🔍 自動掃描章節
掃描整個 OneNote 章節，找出所有行動項目，不遺漏。
📊 清晰表格呈現
以表格方式呈現，含任務說明和目前狀態，一目了然。
✅ 快速確認進度
立刻看出哪些已完成、哪些還在處理中、哪些尚未開始。
📌 適合各種工作場景
專案管理、客戶追蹤、課程筆記都適用，隨時整理隨時用。
🚀 直接複製這段 Prompt（繁中版）
打開 OneNote，把下面整段貼上去就能跑。
請以表格方式摘要本章節中所有我尚未完成的行動項目。為每個行動項目加上說明和狀態。
💬 English Prompt｜英文版
Summarize in a table all the actions I have outstanding across this section. Add a description and a status for each action.
💡 三個情境式玩法
1
📖 週五確認本週行動項目
快速回顧這週所有 OneNote 筆記，確認哪些待辦還沒完成，不讓任何事情漏掉。
2
🔍 接手新專案快速掌握任務
新加入一個專案，透過這個 Prompt 一次看清前人留下的所有未結清行動項目。
3
📊 向上匯報任務執行狀況
主管想了解進度？用這張表格清楚呈現各任務狀態，省去整理時間。
