📸 掃描文件一鍵轉數位文字
Transcribe Scanned Documents Instantly
✨ 手寫＋印刷文字通通數位化
桌上那疊老舊紙本文件或工作坊後的白板便利貼，一直沒有數位化？這個 Prompt 讓 Copilot 識別並轉錄掃描文件或截圖中的所有可見內容，不管是印刷文字還是手寫筆記，通通變成可編輯的數位文字。記得用「上傳文件」按鈕上傳你的圖片或文件喔！紙本文件數位化，一鍵搞定 💾
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat - Web 即可使用
Copilot，請轉錄這份掃描文件的文字。文件中包含手寫筆記和印刷文字，我想將所有相關內容轉換成可編輯的數位文字。
  📋 Prompt Template (English)
Copilot, transcribe the text from this scanned document. The document includes handwritten notes and printed text. I want to transform all relevant content in editable, digital text.
💡 使用小撇步
1. 照片品質越清晰，轉錄結果越準確。盡量確保光線充足、角度正、沒有陰影。
2. 轉錄完成後，可以問 Copilot：「請幫我整理這份文字，格式化成結構清晰的文件。」
3. 若文件有特殊術語或縮寫，可以加上：「以下是文件中可能出現的專業術語：___，請保留原文。」
4. 轉錄後的文字可直接貼到 Word、OneNote 或 Teams，繼續用 Copilot 進行摘要或分析。
