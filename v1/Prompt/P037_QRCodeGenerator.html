💬 QR Code 快速產生器
QR Code Generator
✨ 不需工具，直接問 Copilot 產生 QR Code
開會要分享連結、活動要引導掃碼、名片要放個人頁面……每次需要 QR Code 都要另開分頁找工具，有些工具還有浮水印或廣告，在重要場合用起來很不專業。

QR Code 產生 其實可以直接在 Copilot Chat (Work) 完成！只要一句話告訴 Copilot 你想要的網址，它就能立刻幫你產生可直接掃描的 QR Code。

無論是 簡報中插入連結、活動現場引導，還是 名片數位化，這個超簡單的 Prompt 讓你在 30 秒內完成，不再中斷工作節奏！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat 即可使用
請為 [我的網站 URL] 產生一個 QR Code。
  📋 Prompt Template (English)
Please create me a QR code for [mywebsite]
💡 三個情境玩法
1. 📊 簡報中插入 QR Code
在 PowerPoint 簡報中加入 QR Code，讓觀眾直接掃描跳至補充資源，提升互動效果。
2. 🎫 活動現場快速引導
報名頁、資訊頁一秒轉成 QR Code，印出即可張貼，省去輸入網址的麻煩。
3. 💼 名片數位化升級
把個人 LinkedIn、個人作品頁面轉成 QR Code，名片上多一個科技感亮點。
