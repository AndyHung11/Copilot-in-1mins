本 期 PROMPT
🎓 Copilot 技能學習導師
Your Personal Skill Learning Coach
✨ 技巧＋資源＋練習，零基礎也能上手
想精通一個新方法或技能，卻不知道從何下手？這個 Prompt 把 Copilot 變成你的專屬學習教練！只要告訴它你想學什麼，它就會給你實用技巧、推薦學習資源，還有具體練習，幫你真正學會並應用。就算是零基礎也完全沒問題！想學什麼，就學什麼，就在現在 🚀
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat - Work 即可使用
[方法/技能/關鍵詞]：幫我學會這個技能或方法。提供技巧、資源和實際練習，幫助我在這個領域進步。摘要核心概念，並提供清晰易懂的說明，幫助我理解並有效應用這個概念。
  📋 Prompt Template (English)
[Method/Skill/Keyword]: Help me learn this skill or method. Provide tips, resources, and practical exercises that will help me improve in this area. Summarize the key concepts and offer a clear, simplified explanation to help me understand and apply the idea effectively.
💡 使用小撇步
1. 把 [方法/技能/關鍵詞] 換成你真正想學的東西，越具體效果越好。例如：「OKR 設定」或「用 Python 做資料視覺化」。
2. 從練習任務中選一個最簡單的立刻做，不要只是讀完就關掉。
3. 學習過程中有任何不懂，直接問 Copilot：「請用更簡單的方式解釋 ___。」
4. 建議把 Copilot 推薦的學習路徑存到 OneNote，作為你的個人學習計畫追蹤進度。
