🌱 優雅說不：PLANT 拒絕框架
Graceful No with PLANT Framework
✨ 適用平台：Copilot Chat - Web
職場上最難說的字之一就是「不」——說了怕傷感情，不說又把自己壓垮。

PLANT 框架（暫停 Pause、聆聽 Listen、評估 Assess、命名 Name、轉化 Translate）是一套幫助你有條理、有尊嚴地說不的方法。這個 Prompt 讓 Copilot 根據你的具體情境，套用這套框架，生成一份尊重對方、同時清楚保護自己的回應。

再也不需要臨場結巴，讓 Copilot 先幫你想好每一句話！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat - Web 即可使用
我需要幫助為一個職場情境擬定有效的「拒絕」回應。身為 [你的職務]，我被要求 [描述具體請求]，但我有顧慮，因為 [說明你的具體疑慮]。請運用 PLANT 框架（暫停 Pause、聆聽 Listen、評估 Assess、命名 Name、轉化 Translate），幫我撰寫一份回應，能：1. 尊重地承認請求 2. 清楚說明我的具體顧慮 3. 解釋影響或風險 4. 提出建設性替代方案或妥協 5. 預測可能的反對意見並準備回應。請提供我在這情境中可使用的具體措辭。
  📋 Prompt Template (English)
I need help formulating an effective "no" response for a work situation. As a [your role], I've been asked to [describe specific request], but I'm concerned because [explain your specific hesitation or concern]. Using the PLANT framework (Pause, Listen, Assess, Name, Translate), help me craft a response that: 1. Acknowledges the request respectfully 2. Clearly identifies my specific concern 3. Explains the impact or risk 4. Offers a constructive alternative or compromise 5. Anticipates potential pushback and how I might respond. Please provide me with specific language I could use in this situation.
💡 使用小撇步
1. 📝 填入具體情境
把 [你的職務]、[具體請求] 和 [疑慮] 替換成你的真實狀況，產出才會精準。
2. 🎭 角色扮演測試
請 Copilot：「模擬對方可能的反應，然後幫我練習如何回應。」
3. 📋 調整語氣
如果對象是直屬主管，加上「請讓語氣更尊重且循序漸進」讓回應更恰當。
