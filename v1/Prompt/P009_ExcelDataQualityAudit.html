📊 Excel 資料品質稽核與修復清單
Excel Data Quality Audit
✨ 分析前先清資料，一個 Prompt 找出所有問題並給你修復指南
在 Excel 裡做分析，最怕的就是資料根本是「髒的」——有重複的列、異常的數字、格式亂七八糟的類別——但你渾然不知，做出來的報告結論全是錯的。

這個 Prompt 讓 Copilot in Excel 幫你全面稽核工作表上的資料，找出重複值、異常值、空白、不一致的格式，還提供 KPI 快照和修復清單。不需要巨集、不需要外掛，只要 Copilot 和你的資料，就能做出品質保證的分析。
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Excel 即可使用
任務：對這份工作表上的資料表進行概況分析與品質檢查。
假設：第一列為標題。若有多個資料表，請先問我使用哪一個。若需要 ID 或數值欄位，請先詢問我。
請對資料的副本執行以下操作：
1. 資料概況：每個欄位的資料類型、空白率、唯一率、最小/最大值（數值或日期欄位）
2. 品質標記：重複值（以最佳判斷鍵）、異常值（數值z-score>3）、不一致類別（大小寫/空格變體）、日期缺口
3. KPI 快照：針對選定數值欄位建立基礎統計表（筆數、均值、中位數、標準差）
4. 修復清單：列出問題、影響筆數及建議修法（公式或步驟）
若有任何必要欄位不明確，請在繼續前先詢問。說明請保持簡潔。
  📋 Prompt Template (English)
Task: Profile and quality-check the table on this worksheet. Assumptions: The first row contains headers. If multiple tables exist, ask me which one to use. If you need an ID or numeric column, ask me first. Do the following with a copy of the data: Create a Data Profile — for each column: data type, % blank, % unique, min/max (if numeric/date). Quality Flags — list duplicates (by a best-guess key), outliers (z-score > 3 for numeric), inconsistent categories (case/spacing variants), and date gaps. KPI Snapshot — a separate table basic stats for a chosen numeric column (row count, mean, median, standard deviation). Fix-It Checklist — a table listing issue, count impacted, and a suggested fix (formula or step). Ask clarifying questions before proceeding if any required column isn't clear. Keep explanations concise.
💡 使用小撇步
📂 上傳 Excel 檔案：在 Copilot in Excel 中直接對目前工作表執行，或上傳檔案給 Copilot Chat。
🔎 跑完後追問細節：說「展開第2欄的重複值清單，顯示每筆重複的列號」取得更具體資訊。
📊 資料量大時拆分執行：若資料表超過10000列，可先針對特定欄位執行概況分析。
