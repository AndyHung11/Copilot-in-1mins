本 期 PROMPT
💬 文件最佳實踐差距分析
Document Best Practices Gap Analysis
✨ 文件交出前，先讓 Copilot 幫你找漏洞
文件寫完了，但你心裡還是有個疙瘩——有沒有哪個重要部分漏掉了？有沒有符合公司的 規範要求？

這個 Prompt 在 Copilot Chat (Work) 中執行，你上傳 你的文件 和 最佳實踐指南（例如公司規範、業界標準或產品要求），Copilot 就會幫你做完整的差距分析，列出 尚未涵蓋的要點 和 可以進一步改善的地方。

無論是 產品規格書、提案文件，還是 流程說明書，交出前先讓 Copilot 幫你審查一遍，讓文件品質更有保障！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat 即可使用
請檢視我的 [document.docx]，並對照此處列出的最佳實踐 [Best-Practices.pdf] 進行分析。列出我們尚未涵蓋的內容，以及還可以進一步改進的地方。
  📋 Prompt Template (English)
Review my [document.docx] and analyze it against the best practices listed here [Best-Practices.pdf]. Make a list of the things that we do not yet cover and the things where we can still improve.
💡 三個情境玩法
1. 📐 產品規格書審查
確認技術文件完整符合公司規範與業界標準，交出前有充分把握。
2. 📤 提案文件升級
在送出前確認提案涵蓋所有關鍵要素，提高被採納的機率。
3. 🔄 流程文件稽核
對照品質標準找出流程文件的遺漏與改善點，確保流程文件始終保持最新。
