會議結束後，整理跟進信是最讓人頭痛的事之一。這個 Prompt 根據會議逐字稿，自動產出一封完整的跟進信：包含決議事項表格、行動項目與負責人表格，以及下次會議的待討論清單。複製一鍵寄出，1 分鐘搞定 ✉️
🎁 你會得到什麼？
📋 決議事項表格
清楚列出會議中所有確認的決議，格式整齊，方便後續查閱。
✅ 行動項目與負責人
自動整理每個行動項目及其負責人，確保責任清楚，不遺漏。
📌 下次會議議題清單
列出下次會議需要追蹤或討論的考量事項，提前準備。
📧 專業郵件格式
產出可直接使用的完整跟進信，格式專業，直接複製寄出。
🚀 直接複製這段 Prompt（繁中版）
打開 Teams，把下面整段貼上去就能跑。
請根據 [會議逐字稿] 為與會者撰寫一封跟進信。信件應包含：一張呈現會議決議事項的表格；一張列出行動項目及各項目負責人的表格；以及下次會議的注意事項清單。
💬 English Prompt｜英文版
Write a follow-up email to the attendees of this meeting based on the [Meeting Transcript]. The email should include: A table showing the decisions made during the meeting. A table listing the actions to be taken, along with the owners responsible for each action. A list of considerations for the next meeting.
💡 三個情境式玩法
1
📋 重要會議後立刻發跟進信
高層會議一結束，馬上產出包含決議和行動項目的正式跟進信，展現專業執行力。
2
✅ 確保行動項目不遺漏
逐字稿很長，手動整理容易漏掉某個行動項目，讓 Copilot 幫你全面把關。
3
📁 建立正式會議紀錄存檔
跟進信同時作為正式會議紀錄，格式統一、內容完整，方便未來查閱。
