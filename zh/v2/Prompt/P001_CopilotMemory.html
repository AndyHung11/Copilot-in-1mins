正事還沒問
Copilot Memory 是新功能 — 它會記住你的偏好、工作風格、關鍵細節，讓對話更有效率、更貼近你的情境。這週的 Prompt 就是 啟動鑰匙 — 它會問你三個簡單的問題，然後一次把答案存進你的 Copilot Memory 個人檔案。一次設定，從此告別重複交代 🌿
🎁 你會得到什麼？
💼 工作風格 Memory
你怎麼安排一天、喜歡先看摘要還是細節、會議節奏 — Copilot 都記住。
✍️ 寫作風格 Memory
慣用語氣、長度、稱呼方式（Hi vs 您好），下次寫信不用再修。
🎯 角色 Memory
職稱、團隊、客戶類型 — 讓 Copilot 用對你最有用的角度回答。
🔧 隨時可改
在 Copilot 設定 ➜ Memory 隨時檢視、刪除、新增任何一條記憶。
🚀 直接複製這段 Prompt（繁中版）
打開 Copilot Chat (Web 或 Work)，把下面整段貼上去就能跑。
幫我啟動我的 Copilot Memory。請問我三個簡單的問題：分別關於我的工作風格、寫作風格，以及我的職務角色的關鍵事實。請把我的回答加到我的 Copilot Memory 個人檔案中。
💬 English Prompt｜英文版
Help me with getting started on my Copilot Memory. Ask me three simple questions about my work style, writing style and key facts about my role. Add the results to my Copilot Memory profile.
💡 三個情境式玩法
1
🔍 第一次跑完馬上驗收
問 Copilot：「請複述你剛剛幫我記住的每一條 Memory」確認它有完整收到。
2
➕ 隨時加新記憶
日常對話中說「Copilot, 我比較偏好…」或「請記住…」就會看到小通知，直接存進 Memory。
3
🧹 每季回頭整理
打開 Copilot 設定 ➜ Memory，刪掉過時或不再適用的舊偏好，保持精準。
