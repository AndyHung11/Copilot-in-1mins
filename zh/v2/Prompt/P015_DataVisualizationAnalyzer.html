你有一份財務、銷售或行銷資料，但打開 Excel 後卻不知道從哪開始分析？或者你做出了一堆圖表，卻說不清楚「所以呢？數字背後的故事是什麼？」

Analyst Agent 讓你上傳一個或多個資料檔案，它會自動分析、視覺化，並找出哪些指標表現好、哪些需要關注，以及是什麼在驅動這些變化。你可以把「areas」替換成任何 KPI，例如產品、服務、客戶區段或行銷活動。
🎁 你會得到什麼？
📊 自動視覺化呈現
自動選擇最適合的圖表類型，讓資料一目瞭然。
✅ 表現優異與需改進區域
清楚標出哪些指標/產品/地區表現最好或最差。
🔍 主要驅動因素分析
找出影響數字變化最大的關鍵因素，不再靠猜測。
🌏 地區/細分市場深入分析
若資料允許，進一步分析地區或客戶細分績效。
🚀 直接複製這段 Prompt（繁中版）
分析並視覺化這份資料，找出表現良好的 [指標/區域] 和表現不佳的 [指標/區域]。什麼因素對這些變化影響最大？另外，如果資料允許，請分析細分市場或地區績效。
📋 Prompt Template (English)
Analyze and visualize this data and identify [areas] that are doing well and [areas] that are not doing so well. What is influencing changes the most? Additionally, if data is available, analyze segment or regional performance.
💡 三個情境式玩法
1
📎 上傳資料檔案：在 Analyst Agent 中上傳 Excel、CSV 或其他資料檔案，再貼上這個 Prompt。
2
🔄 替換 [指標/區域]：可以換成「產品線」、「銷售地區」、「行銷活動」等你實際關注的維度。
3
🔍 深入追問：收到初步分析後說「請針對表現最差的地區展開原因分析」取得更深入的洞察。
