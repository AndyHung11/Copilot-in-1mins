🎮 學習闖關徽章三連發

幫我設計一張「升級打怪」風格的徽章圖，包含三個闖關等級，由弱到強分別是 🌱 [新手]、⚔️ [勇者]、🏆 [傳說]，呈現一種「從菜鳥一路打到大神」的成長感。

三顆徽章要像同一套遊戲成就系統：相同的外框形狀與整體設計，每顆上方都放上活動名稱 [你的團隊／課程名稱]。隨著等級越高，配色越濃烈、圖示越華麗（例如新手清新、勇者熱血、傳說閃耀奪目），但整體風格維持一致、一看就是一系列。

請讓三顆徽章並排、彼此不重疊，每顆都有完整邊緣不被裁切，整體乾淨好看、可直接當獎章使用。
📋 Prompt Template (English)
🎮 Level-Up Quest Badges

Design a set of "level-up / quest" style badges with three ranks, going from weakest to strongest: 🌱 [Newbie], ⚔️ [Adventurer], 🏆 [Legend] — capturing a fun zero-to-hero progression.

The three badges should feel like one game achievement system: the same outline shape and overall design, each with the program name [Your Team or Course Name] across the top. As the rank climbs, make the palette richer and the icon fancier (e.g. Newbie fresh and light, Adventurer bold and fiery, Legend dazzling and glowing), while keeping one consistent style so they clearly read as a series.

Place the three badges side by side, without overlapping, each with a full, uncropped edge, clean and ready to hand out as awards.
💡 三個情境式玩法
1
🎨 在 Create 分頁貼上並改名
Copilot →「Create（建立）」分頁貼上整段 Prompt，把 [你的團隊／課程名稱] 換成你的活動名，三個等級也能自己玩（例如 [打工仔]／[扛霸子]／[天花板]、或 [菜雞]／[老司機]／[大神]）。
2
🔧 針對單顆徽章微調
覺得最高階不夠霸氣？再回一句「把『傳說』那顆改成燃燒金色火焰並加上桂冠」，局部精修不用重畫整組。
3
📥 去背下載即貼上
下載透明背景 PNG，放進 Teams 表揚貼文、簡報封面或 SharePoint 學習頁，頒獎瞬間儀式感拉滿。
