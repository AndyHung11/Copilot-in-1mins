Copilot Chat 現在在 Outlook on the Web 開放給所有人使用（不需 M365 Copilot 授權）！碰到一條長長的郵件串，不知道從哪讀起？

這個 Prompt 幫你用兩句話抓出重點、標出哪些任務是指派給你的，還順手生成一份回覆草稿，讓你快速確認後就可以送出。從此郵件再長也不怕，處理速度直接翻倍。
🎁 你會得到什麼？
✏️ 兩句話核心摘要
快速掌握郵件串的關鍵訊息，不必從頭逐字閱讀。
✅ 我的任務清單
自動標出郵件串中指派給你的所有待辦事項。
📝 回覆草稿自動生成
根據郵件內容產生草稿，確認後即可送出。
🚀 直接複製這段 Prompt（繁中版）
用兩句話摘要這封郵件。在下一節中，標示郵件串中指派給我的任何任務。然後為我的回覆建立一份草稿。
📋 Prompt Template (English)
Summarize this mail in two sentences. In the next section, highlight any task given for me in the mail thread. Then create a draft for my response.
💡 三個情境式玩法
1
📌 在 Outlook 網頁版中使用：選取郵件，點擊 Copilot 圖示，直接貼入這個 Prompt 即可執行。
2
✍️ 草稿是起點，不是終點：記得根據情境調整語氣與細節，加入你的個人風格。
3
🔍 追問更多資訊：說「哪封信對我影響最大？」讓 Copilot 幫你排出回應優先順序。
