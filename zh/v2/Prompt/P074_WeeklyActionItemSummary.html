上週交辦你的事散落在 Email、會議逐字稿、Teams 訊息、檔案註解裡，週一一打開電腦根本不知道從哪開始？

這個 Prompt 讓 Copilot 把所有「指派給我」「@我」「請你跟進」的事項一次撈出來，整理成「事項 / 指派人 / 截止日 / 狀態」四欄的表格。把它設定成 Scheduled Prompt，每週一早上 8:30 自動跑，開工前清單已經躺在信箱裡。
🎁 你會得到什麼？
📋 一張清單看完所有 Action
事項、指派人、截止日、狀態四欄，一目瞭然。
🔄 跨來源整合
Email + 會議逐字稿 + Teams + 檔案 @mention 一次撈完。
⏰ 自動排程跑
設成 Scheduled Prompt，每週一自動產出寄到信箱。
📌 直接接 Planner / To Do
表格產出後直接複製到任務管理工具追蹤。
🚀 直接複製這段 Prompt（繁中版）
打開 Copilot Chat (Work)、Outlook 或 Teams，把下面整段貼上去就能跑。
請彙整上週透過 電子郵件、會議逐字稿、Teams 訊息、以及我被 @ 提及的檔案註解 中所有指派給我的待辦事項。包含任何別人提到的任務、後續追蹤請求或截止日期。請建立一張表格，包含以下欄位：1) 待辦事項，2) 由誰指派，3) 請求的截止日，4) 狀態（若有提到）。若沒有提到截止日，欄位請留空。
💬 English Prompt｜英文版
Summarize all action items from last week, that were assigned to me via email, meeting transcripts, Teams messages, and file comments where I was @mentioned. Include any tasks, follow-up requests, or deadlines that people mentioned. Create a table with the following columns: 1) Action Item, 2) Who Assigned It, 3) Requested Due Date, and 4) Status (if mentioned). If no due date is mentioned, leave it blank.
💡 三個情境式玩法
1
⏰ 設成 Scheduled Prompt 每週一跑
用 aka.ms/ScheduledPrompt 排成每週一 8:30，開工前清單已寄到信箱。
2
📌 一鍵接到 Planner / To Do
產出表格後直接複製到 Microsoft To Do 或 Planner，立刻變成可追蹤任務。
3
🔍 換時間範圍對應不同檢視
把「上週」改成「過去 14 天」「本月」「本季」就能對應週、雙週、月、季的回顧節奏。
💬 範例提示詞｜繁體中文版
請彙整本月指派給我但尚未完成的任務，分組依「優先級高 / 中 / 低」排序，並標出已逾期項目。
💬 Example Prompts｜English Version
Summarize this month's open action items assigned to me, grouped by priority (high/medium/low), and flag any items that are overdue.
