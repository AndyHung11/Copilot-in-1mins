你有沒有看過那種很酷的文字雲視覺化，大大小小的字飄在一起，一眼就看出哪些主題最重要？

這個 Prompt 讓你把任何文件上傳給 Copilot Chat - Web，只要一句話就能生成你自己的文字雲。無論是會議記錄、研究報告、客戶回饋還是年度報告，幾秒鐘後就能看到核心主題一覽無遺。不需要任何設計工具，也不需要 M365 授權。

☁️ 文字雲視覺化圖片
直觀呈現文件最重要的關鍵字，可直接加入簡報或報告。
🔤 前10大關鍵字提取
自動識別並列出最具代表性的詞彙。
🚫 過濾填充詞
自動去除無意義的常用詞，聚焦真正有意義的主題詞彙。
📁 多格式支援
支援 Word、PowerPoint、Excel、PDF 及純文字檔案。
🚀 直接複製這段 Prompt（繁中版）
打開 Copilot Chat (Web 或 Work)，把下面整段貼上去就能跑。
建立 [text-document.docx] 中前10大關鍵字的文字雲。請忽略填充詞。
📋 Prompt Template (English)
Create a Word Cloud of the top 10 key words in [text-document.docx]. Ignore filler words.
💡 三個情境式玩法
📎 上傳文件再執行：把 Word、PDF、PowerPoint 或 Excel 檔案上傳給 Copilot Chat (Web) 後再貼上這個 Prompt。
🔢 調整關鍵字數量：把「前10大」改成「前20大」可以獲得更豐富的文字雲。
🌐 加入語言指定：若文件是英文，追問「請也顯示中文翻譯」讓結果更易讀。
