🧠 告別死背！為你的大腦量身打造備考計畫
Neurodiversity-Friendly Exam Prep Plan
✨ 每個大腦都有最適合自己的學法，讓 Copilot 幫你找到 ✨
大多數教材是為「一般學習節奏」設計的——密密麻麻的文字、缺乏結構的條列，讓很多人讀了半天還是什麼都沒吸收。

這個 Prompt 打破這堵文字牆：先抓出關鍵事實，指示詞簡短且有結構，讓備考變得更清晰、更有節奏。不管你是準備 Microsoft 認證、語言考試還是職業資格，都可以套用。只要填入你的考試資訊，Copilot 就幫你生成一份 完全個人化的備考路線圖。
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat（Web）即可使用
我正在準備 [考試名稱/代碼] 的考試，考試日期是 [日期]。我的程度：[新手 / 中階 / 熟悉]。目標：[通過 / 達到 X 分]。可用時間：[每天 X 分鐘，每週 Y 小時]。適合我的學習方式：[短段落、類比說明、測驗問題…]。對我困難的方式：[大量文字、缺乏結構…]。請只使用免費官方資源（例如 Microsoft Learn）。如有需要，請先問我問題。接著：給我一份 30 天（或按我的時間安排）的計畫，每天 3 個任務，包含間隔複習與休息日。每個概念請用一個容易記憶的故事說明，加上白話解析和練習問題。並加入快速記憶法、10 字重點摘要，以及 60 秒複習腳本。語言：繁體中文，清楚標題，條列式編號。
  📋 Prompt Template (English)
I'm taking an exam / certification [exam name/code] on [date]. My level: [new / mid / confident]. Goal: [pass / %]. Time I have: [x mins/day, y hrs/week]. Works for me: [short chunks, analogies, quizzes, …]. Hard for me: [walls of text, no structure, …]. Use only free official resources (e.g., Microsoft Learn). Ask me anything else you need. Then: give me a 30-day (or my timeline) plan with 3 tasks/day, spaced review, rest days. Explain each concept with one memorable story, plus plain-English breakdown, plus practice questions. Add quick mnemonics, a 10-word summary, and a 60-second recap script. Language: English, clear headings, numbered bullets.
💡 使用小撇步
1
🎓 填入你的真實考試資訊
把 [考試名稱] 和 [日期] 換成具體資訊，Prompt 越具體，計畫越精準。
2
🔁 照著每日 3 任務執行
間隔複習日特別重要，不要跳過，這是記憶鞏固的關鍵。
3
📝 用 60 秒腳本自我測試
睡前把腳本大聲念一遍，對長期記憶有顯著加成效果。
