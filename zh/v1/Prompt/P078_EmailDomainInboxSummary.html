本 期 PROMPT
📥 信件網域摘要
Email Domain Inbox Summary
✨ 一個 Prompt 看完所有客戶來信＋誰已回
客戶下午突然問：「上週那個案子我們同事有沒有回？」——你開始翻郵件、跨過 8 個對話串、查同事副本、確認誰回了什麼。10 分鐘過去，還沒給答案。

這個 Prompt 直接在 Outlook Copilot 用：指定一個網域（例如 @customer.com）與時間範圍（例如過去 4 週），就回一張表——誰寄的、何時寄的、需要什麼行動、我方有沒有人回過。

對於客戶經理、PM、業務、Account 角色超實用：每天開工前跑一次，馬上知道哪些 thread 還沒人接、哪些已經處理掉，把「漏接客戶來信」這個風險直接消掉。
  📋 提示詞範本（繁體中文版）
建立一張表格，列出我在過去 X 週內從 DOMAINNAME（例如 @customer.com）收到的所有 email。請包含一欄顯示這封信需要哪些行動，以及另一欄顯示我公司是否有人已經回覆。
  📋 Prompt Template (English)
Create a tableshowing all emails I received from DOMAINNAME (e.g., @customer.com) in the last X weeks. Include a column that shows any action required from the message and another column to show if anyone at my company has responded yet.
💡 使用小撇步
1. 在 Outlook 的 Copilot 直接貼上（行動裝置或桌面版皆可）。
2. 把 DOMAINNAME 換成實際網域（例如 @chunghwa.com.tw），X 週換成你要的範圍（1、2、4、8…）。
3. 看到「尚無人回覆」的列就先處理；看到「已回覆」的列再確認回得對不對、要不要追加。
