🎨 圖片主色調提取工具
Dominant Color Extractor
✨ 上傳任何圖片，一秒取得 RGB 和 HEX 色碼，搞定設計配色
你有沒有看到一張很漂亮的圖片，想把它的配色用在你的簡報或設計上，但根本不知道那些顏色的確切色碼？或者你需要確保品牌視覺一致性，但手邊沒有設計工具？

這個 Prompt 讓你把任何圖片或截圖上傳給 Copilot Chat - Web，立刻取得最主要的8種顏色，包含 RGB 數值和 HEX 色碼，以及每個顏色的描述標籤（支援無障礙設計）。完全不需要設計工具，也不需要 M365 授權。
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat - Web 即可使用
提取這張圖片中的 [8] 個主要顏色。提供每個顏色的 RGB 數值、HEX 色碼，以及每個顏色的簡短描述（例如用於無障礙設計或設計用途）。
  📋 Prompt Template (English)
Extract the [8] dominant colors from this image. Provide the RGB values, HEX codes, and a short description of each color (e.g., for accessibility or design purposes).
💡 使用小撇步
📸 上傳圖片再執行：把圖片或截圖直接上傳到 Copilot Chat (Web)，再貼上這個 Prompt。
🔢 調整提取數量：把 [8] 改成 [5] 或 [12]，根據需要提取更少或更多的顏色。
🎯 指定顏色用途：說「這些顏色用於行銷素材配色」讓描述標籤更貼合你的應用場景。
🆓 不需 M365 Copilot 授權，Copilot Chat (Web) 即可使用
使用平台：Copilot Chat - Web
