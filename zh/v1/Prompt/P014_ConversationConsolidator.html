🔄 對話彙整為單一可重用提示詞
Conversation Prompt Consolidator
✨ 多輪調整後的完美對話，一鍵萃取成可重複使用的提示詞
你有沒有花了好幾輪對話，終於讓 Copilot 給出完美的回答——但你根本不知道怎麼重現它？下次想做類似的事，又要從頭慢慢試，真的很浪費時間。

這個 Prompt 讓 Copilot Chat 把整段對話濃縮成一個單一的、可重複使用的提示詞，讓你下次直接貼上就能拿到相同品質的輸出。特別適合用來建立你自己的提示詞庫，或自動化那些每週都要重複的例行性任務。
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat - Web 即可使用
摘要這個對話串中的完整對話，並生成一個單一的、完整的提示詞，可以重現我收到的最新回覆。目標是將所有之前的提示和情境整合成一個可重複使用的輸入。
  📋 Prompt Template (English)
Summarize the entire conversation so far in this thread and generate a single, comprehensive prompt that would reproduce the latest response I received. The goal is to consolidate all previous prompts and context into one reusable input.
💡 使用小撇步
💾 立刻儲存結果：跑完後把生成的提示詞複製存到筆記或文件，建立你的個人提示詞庫。
🔧 微調後再存：收到整合提示詞後，可以稍作調整讓它更通用，適用於更多情境。
👥 分享給團隊：把好用的提示詞分享給同事，讓整個團隊都受益於你調整出來的最佳方案。
