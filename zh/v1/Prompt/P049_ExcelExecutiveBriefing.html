💬 Excel 一頁式主管決策簡報
Excel One-Page Executive Briefing
✨ 把密密麻麻的 Excel 變成主管一眼看懂的決策工具
主管說：「給我看一下這季的數字。」你打開了一個有 20 個分頁、5,000 行資料的 Excel……然後沉默了 30 秒。

這個 Prompt 在 Analyst Agent 中執行，你只需上傳 Excel 檔，Copilot 就會自動摘要成一頁式主管決策簡報：包含 KPI 摘要、趨勢分析、警示紅旗，以及自動選擇最適合的 視覺化圖表，讓複雜的數據關係一目了然。

無論是 季度業績報告、銷售漏斗分析，還是 專案健康狀態追蹤，讓 Copilot 幫你把數字變成主管能直接帶進會議室的決策工具！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat 即可使用
請將此 Excel 檔整理成一頁式主管決策簡報，包含 KPI、趨勢與任何警示紅旗。請提供一個合適的資料集視覺化圖表（例如長條圖或圓餅圖）。此圖表應幫助使用者快速發現相關性與趨勢，進一步強化一頁簡報的效果。
  📋 Prompt Template (English)
Summarize this Excel file into a one-page executive briefing. Include KPIs, trends, and any red flags. Give me a suitable visual representation of the dataset in a graph of your choice - for example, a bar or pie chart. This should help the user quickly notice correlations and trends, adding more complement to the one-pager's intent.
💡 三個情境玩法
1. 💼 季度業績報告
把密密麻麻的財務數字轉化成一張主管可以帶進董事會的精華摘要。
2. 📣 銷售漏斗分析
從轉換率數字中自動找出瓶頸並視覺化呈現，快速定位需要強化的環節。
3. 📡 專案狀態追蹤
將多個指標整合成一目了然的專案健康狀態圖，讓所有人共享同一份數據視角。
