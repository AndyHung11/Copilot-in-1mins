Multi-Document FAQ Generator
✨ 把一疊文件濃縮成一份 FAQ，讓每個人都能快速上手
你有一份詳細的產品規格簡報和一份設計文件……但新來的工程師說他不知道從哪裡讀起，客戶說找不到他要的資訊……每次都要手動回答同樣的問題，真的很累。

這個 Prompt 在 Word 中執行，你可以同時連結 多份文件（簡報、報告、會議記錄等），Copilot 會自動整合內容，產出一份包含 清楚標題、條列點，並附上完整解答的 FAQ 文件。

輸出支援 Markdown 格式，可直接貼到 Teams 聊天、SharePoint 或 Wiki 頁面，讓每個人都能自助找到答案，從此告別重複回答同樣問題！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat（Web）即可使用
請依據 [presentation.pptx] 和 [document.docx] 建立一份 FAQ，包含清楚的標題與條列點以提升可讀性。確保文件結構能有效回應常見問題，並為目標讀者提供完整的解答。考慮使用 Markdown 格式化文件，方便分享與分發。
  📋 Prompt Template (English)
Create an FAQ based on [presentation.pptx] and [document.docx] that includes clear headings and bullet points to enhance readability. Ensure that the document is structured in a way that addresses common questions effectively and provides comprehensive answers for the intended audience. Consider using Markdown to format the document for easy sharing and distribution.
