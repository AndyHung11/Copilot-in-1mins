🧩 問題解決腦力激盪夥伴
Your Problem-Solving Brainstorm Buddy
✨ 迭代提問＋量身解法，突破思考盲點
碰到棘手問題，自己想不到解法，問別人又得不到有深度的建議？這個 Prompt 把 Copilot 變成你的問題解決夥伴！它先了解你的問題，再問你三個最關鍵的問題來深挖，然後提出量身打造的解法 — 而且每輪都追加三個新問題繼續迭代，幫你從多個角度突破盲點。問題越難，越要有好夥伴 🧩
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat - Web 即可使用
我需要你成為我的問題解決腦力激盪夥伴。請先問我有什麼問題或機會。然後提出 3 個最相關的問題，幫助我找到最佳解決方案。根據我的回答，提出解決我挑戰的方案，並每次都附加 3 個新問題來持續迭代解法。
  📋 Prompt Template (English)
I need you to become my problem-solving brainstorm buddy. Start by asking me about my problem or opportunity. Then ask me the 3 most relevant questions to help me find the best solution. Based on my answers, propose a solution to my challenge and always add 3 new questions to iterate on the solution.
💡 使用小撇步
1. 一開始只要說出問題的大方向就好，讓 Copilot 的問題引導你說清楚細節。
2. 回答 Copilot 的問題時，盡量具體，越多真實背景資訊，解法越量身訂製。
3. 覺得某個解法方向有趣，可以說：「請針對這個方向再深入探討」，繼續迭代。
4. 這個 Prompt 適合任何類型的問題：工作挑戰、策略規劃、人際關係，甚至創意發想都很好用。
