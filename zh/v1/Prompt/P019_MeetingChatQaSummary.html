💬 💬 會議聊天室問答摘要文件
Meeting Chat Qa Summary
✨ 把資訊爆炸的會議聊天記錄，整理成結構清晰的問答文件
參加了一場超級精彩的會議，資訊量大到有點消化不過來？大家在聊天室裡問了很多問題、分享了很多資料，但會議一結束，那些寶貴的討論就消失在記錄的洪流中。

這個 Prompt 讓 Copilot Chat - Work 把你的會議聊天記錄轉化成一份結構化的問答文件，使用 Markdown 格式清晰分層，關鍵資訊加粗標示，讓整個團隊都能快速查閱最重要的討論內容。
  📋 提示詞範本（繁體中文版）
根據 [會議聊天記錄] 的會議聊天記錄，生成一份包含最重要問題的完整問答文件。使用 Markdown 格式設計文件，包含每個問題和答案的清晰標題、重點的條列格式，以及粗體標示重要資訊。確保文件視覺清晰且易於導覽。
  📋 Prompt Template (English)
Based on the meeting chat of [meeting chat], generate a comprehensive Q&A document with the most interesting questions. Use Markdown to format the document, including clear headings for each question and answer, bullet points for key points, and bold text to highlight important information. Ensure the document is visually appealing and easy to navigate.
💡 使用小撇步
📋 提供聊天記錄：把 Teams 聊天記錄複製或上傳，替換 [會議聊天記錄] 後執行。（若改讓 Copilot 直接讀取 Teams 會議則需進階授權）
🎯 指定關注的問題類型：加上「特別關注技術問題和待辦事項」讓文件更聚焦。
📤 分享給未出席者：問答文件是讓沒有出席的同事快速跟上的最佳工具。
