💬 Excel 數值相關性分析報告
Excel Data Correlation Analysis
✨ 數字背後的規律，一張報告全看透
你手上有一份包含幾十個欄位的業務報表，每個數字都看過了，但就是說不出「客戶滿意度」和「問題解決率」之間到底有什麼關係……

這個 Prompt 在 Excel 中執行，Copilot 會針對你工作表中所有的數值欄位，自動計算相關係數、摘要 關鍵發現，並根據識別出的相關性提供 潛在行動建議。

真實世界充滿了變數之間的關聯，理解這些關聯是預測趨勢、做出明智決策的第一步。讓 Copilot 幫你從數字堆中找到隱藏的規律與機會！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat 即可使用
請針對數值欄位產生相關性報告，包含關鍵發現摘要，並根據識別出的相關性提供潛在行動建議。
  📋 Prompt Template (English)
Produce a correlation report for the numerical columns. Include a summary of key findings and provide insights on potential actions based on the correlations identified.
💡 三個情境玩法
1. 📈 業務績效分析
找出客戶滿意度與問題解決率的相關性，優化客服流程以提升整體滿意度。
2. 📣 行銷效益評估
分析廣告投入與業績成長的關聯，將預算集中在最有效益的行銷管道。
3. 👥 人力資源洞察
了解員工訓練時數與生產力指標的相關性，優化培訓投資以獲得最大回報。
