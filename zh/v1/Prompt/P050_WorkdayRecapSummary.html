💬 工作日訊息回顧與行動摘要
Workday Recap & Action Summary
✨ 一個表格搞定郵件和 Teams，今天的事今天結清
快下班了，你突然不確定今天有沒有漏掉什麼重要訊息……翻了一下 Outlook，又打開 Teams 看了一下，還是不太確定。

這個 Prompt 在 Copilot Chat (Work) 中執行，Copilot 會同時摘要你上一個工作日的 Outlook 郵件 和 Teams 訊息，整理成一張結構化表格：類型（郵件/Teams）| 主題 | 摘要 | 行動項目 | 後續行動。

如果你被 直接點名，該主題的字體會自動加粗，讓你一眼就看到最需要優先回應的訊息。無論是 每日下班收尾、週一早晨補課，還是 跨時區協作追蹤，都能幫你不遺漏任何重要事項！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat 即可使用
請摘要我上一個工作日的郵件和 Teams 訊息，並在獨立欄位中列出行動項目。如果可以的話，請在另一個獨立欄位中建議後續行動。表格格式如下：類型（郵件/Teams）| 主題 | 摘要 | 行動項目 | 後續行動。如果我被直接點名，請將該主題的字體加粗。
  📋 Prompt Template (English)
Summarize my emails and Teams messages from the last workday. List action items in a separate column. If possible, suggest follow-up actions in a separate column. The table should look as follows: Type (Mail/Teams) | Topic | Summary | Action Item | Follow-up Action. If I was mentioned directly, please make the font of the topic bold.
💡 三個情境玩法
1. 🌙 每日下班收尾儀式
快速確認今日工作狀態，有條不紊地結束一天，帶著清晰的心情離開辦公室。
2. ☕ 週一早晨補課
週末後回來時，迅速掌握週五下班後的訊息動態，快速進入工作節奏。
3. 🌏 跨時區協作追蹤
確認非工作時間收到的跨國訊息中有哪些待辦事項，讓協作不因時差而延遲。
