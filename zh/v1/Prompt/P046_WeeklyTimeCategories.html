💬 每週時間分配類別分析
Weekly Time Allocation Analyzer
✨ 日曆看完才知道，時間真正花在哪裡
你一整天都很忙，但到了週五下班，回頭看這週到底做了什麼……好像什麼都做了，又好像什麼都沒完成。

這個 Prompt 在 Copilot Chat (Work) 中執行，Copilot 會分析你接下來一週的 日曆安排，自動歸納出 五個時間使用類別，每個類別附上說明和估計的 時間佔比。

知道自己的時間都花在哪裡，你才能 有意識地做出調整——增加深度工作的比例、減少低效會議、或重新平衡不同工作類型的時間分配。數據才是時間管理的起點！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat 即可使用
依據我接下來這一週的日曆，建立 五個 描述我如何運用時間的類別，並為每個類別提供簡短說明，以及估計我在每個類別上花費的時間百分比。
  📋 Prompt Template (English)
Looking at my calendar over this coming week, create five categories that describe how I spend my time. Give me a short description for each category and estimate the % time I spend on each.
💡 三個情境玩法
1. 📆 季度時間審視
每季一次，確認時間分配是否還符合當前的工作重點與角色需求。
2. 🆕 新職位或新角色適應
了解新角色的時間組成，找出需要調整的習慣，更快進入狀態。
3. 🧘 提高深度工作比例
找出被零碎事務佔用的時間，重新規劃，為高品質深度工作騰出空間。
