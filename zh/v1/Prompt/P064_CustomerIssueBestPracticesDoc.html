本 期 PROMPT
📚 客戶問題最佳實踐文件生成
Customer Issue Best Practices Document
✨ 適用平台：Word
每個支援團隊都有這樣的痛點：同樣的客戶問題一再出現，但每次解決的方式卻不一樣，品質不穩定，而且所有的「解法」都只存在老鳥的腦袋裡……

這個 Prompt 讓 Copilot 從你與 [人員/團隊] 的討論記錄中，直接生成一份最佳實踐文件——包含系統化的處理流程、過去成功案例的具體範例，以及應對常見客戶問題的技巧清單。

讓知識從個人記憶變成團隊資產！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Word 即可使用
根據與 [人員／團隊] 討論 [主題] 相關客戶問題的溝通記錄，請建立一份更新版的最佳實踐文件，概述解決類似問題的系統化方法。包含來自過去成功解決案例的範例，以及處理常見客戶問題的技巧。
  📋 Prompt Template (English)
Based on the discussion notes with [person / team] regarding the recent customer issue with [topic], create an updated best practices document that outlines a systematic approach for resolving similar complaints. Include examples from previous successful resolutions and tips for handling common customer questions.
💡 使用小撇步
1. 📎 貼上討論記錄
把與相關人員的郵件串、Teams 對話或會議摘要一起貼進來，讓文件有真實依據。
2. 📂 存進 SharePoint
生成後存入團隊共享文件庫，讓所有成員都能隨時查閱。
3. 🔄 定期更新
每季回顧一次，請 Copilot 根據新的案例更新文件，保持內容的時效性。
