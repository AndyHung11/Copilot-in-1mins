本 期 PROMPT
📬 信件主題分類一覽
Categorize Last Week's Emails by Theme
✨ 5 大主題＋佔比，工作重心一眼看清
你知道上週的信件都在聊什麼嗎？這個 Prompt 讓 Copilot 掃描你上週的所有信件，自動歸類成 5 大主題，並給出每個主題佔比的概略百分比。清楚看到自己的工作重心，幫助你重新檢視優先順序，以更聚焦的狀態迎接下一週！了解你的時間分佈，做更聰明的選擇 📊
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Outlook 即可使用
請將我上週的信件分類成 5 大主題。並依各主題的信件數量，給出每個大方向的概略百分比。
  📋 Prompt Template (English)
Categorize my emails from last week into 5 themes. Give a rough percentage weight for each overarching topic based on the number of emails in each theme.
💡 使用小撇步
1. 在 Outlook 中使用此 Prompt，Copilot 能直接存取你的信箱進行分析。
2. 如果想換成「這個月」或「這一季」，直接把「上週」換成對應的時間範圍就好。
3. 看到某個主題佔比出乎意料地高，可以繼續問：「哪些信件屬於這個主題？請列出。」
4. 建議每兩週執行一次，觀察工作重心的變化趨勢，有助於季度回顧與職涯規劃。
