🌐 簡報語言一鍵翻譯
Translate Your Presentation in One Click
✨ 一指令產出完整翻譯版新簡報
需要把整份英文簡報翻成中文，或把中文版翻給外國客戶看？手動一張張翻譯是上個世紀的做法。這個 Prompt 讓 Copilot 自動建立一份全新翻譯版簡報，所有投影片上的文字都翻譯成你指定的語言，原始版本完整保留，翻譯版另存為新文件！一個指令，打開國際市場 🌏
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 PowerPoint 即可使用
請建立這份簡報的新版本，並將所有投影片翻譯成 [選擇語言]。
  📋 Prompt Template (English)
Please create a new version of this slide deck and translate all the slides to [Choose Language]
💡 使用小撇步
1. 把 [選擇語言] 換成你需要的語言，例如：「日文」、「英文」、「西班牙文」、「韓文」等。
2. 翻譯完成後，快速瀏覽一遍，確認專業術語和產品名稱是否翻譯正確（建議保留英文原文）。
3. 如果有特定用詞要求，可以加上：「請保留以下詞彙的英文原文：___」。
4. 翻譯版本適合作為初稿，建議有母語人士快速核閱一遍再正式使用。
