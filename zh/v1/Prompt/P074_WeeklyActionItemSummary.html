本 期 PROMPT
一鍵彙整本週指派給我的待辦事項
Weekly Action Items Summary
✨ 週一早上 Copilot 自動幫你把分散在 Email、會議、Teams、檔案的 Action 撈成一張表 📋
上週交辦你的事散落在 Email、會議逐字稿、Teams 訊息、檔案註解裡，週一一打開電腦根本不知道從哪開始？

這個 Prompt 讓 Copilot 把所有「指派給我」、「@我」、「請你跟進」的事項一次撈出來，整理成有「事項 / 是誰指派 / 截止日 / 狀態」四欄的表格。把它設定成 每週一自動執行的 Scheduled Prompt，每週開工前你的待辦清單就已經躺在信箱裡。
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat (Work)、Outlook 或 Teams 即可使用
請彙整上週透過 電子郵件、會議逐字稿、Teams 訊息、以及我被 @ 提及的檔案註解 中所有指派給我的待辦事項。包含任何別人提到的任務、後續追蹤請求或截止日期。請建立一張表格，包含以下欄位：
1) 待辦事項（Action Item）
2) 由誰指派（Who Assigned It）
3) 請求的截止日（Requested Due Date）
4) 狀態（Status）（若有提到）
若沒有提到截止日，欄位請留空。
  📋 Prompt Template (English)
Summarize all action items from last week, that were assigned to me via email, meeting transcripts, Teams messages, and file comments where I was @mentioned. Include any tasks, follow-up requests, or deadlines that people mentioned. Create a table with the following columns: 1) Action Item, 2) Who Assigned It, 3) Requested Due Date, and 4) Status (if mentioned). If no due date is mentioned, leave it blank.
💡 使用小撇步
⏰ 排程每週一早上自動跑：用 Scheduled Prompts 設成每週一 8:30 自動執行，週一打開信箱就有清單。
📌 把表格貼到 Planner / To Do：產出後直接複製到 Microsoft To Do 或 Planner，一鍵變成可追蹤的任務。
🔍 調整時間範圍：把「上週」改成「過去 14 天」「本月」就能對應不同檢視週期。
  💬 範例提示詞（Example Prompts）
請彙整本月指派給我但尚未完成的任務，分組依「優先級高 / 中 / 低」排序，並標出已逾期項目。
英文版：
Summarize this month's open action items assigned to me, grouped by priority (high/medium/low), and flag any items that are overdue.
