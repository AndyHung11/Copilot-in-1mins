本 期 PROMPT
🎤 簡報演講者備忘稿優化
Speaker Notes Optimizer
✨ 一個 Prompt 幫整份簡報加上清晰易讀的演講者備忘稿
準備一份重要的簡報，但備忘稿要嘛是空的，要嘛寫得太複雜，演講的時候還要分神去看——這種情況你有沒有遇過？

這個 Prompt 讓 Copilot in PowerPoint 幫你為整份簡報每一張投影片加上並優化演講者備忘稿，使用短句和簡單語言，確保你演講時能夠清晰流暢地引導觀眾。無論你在準備團隊會議、客戶簡報還是研討會演講，都能讓你的表現更專業。
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 PowerPoint 即可使用
為整份投影片加上並優化演講者備忘稿。請使用短句和簡單語言，讓觀眾更容易跟上內容。
  📋 Prompt Template (English)
Add and optimize speaker notes for the whole slide deck. Use short sentences and simple language, so it is easy to follow for the audience.
💡 使用小撇步
🎯 指定語氣風格：可以加上「請使用正式語氣」或「輕鬆對話風格」讓備忘稿符合你的演講風格。
📏 指定長度：說「每張投影片備忘稿不超過50字」確保備忘稿簡潔易讀。
🔄 結合提問引導：加上「每張投影片結尾加上一句可以帶動討論的問題」讓演講更有互動性。
✅ 需要 M365 Copilot 授權
使用平台：PowerPoint
