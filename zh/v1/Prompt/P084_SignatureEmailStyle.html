💅 幫每封信設定專屬撰寫風格
Set Your Signature Email Style
✨ 設定一次撰寫風格，之後每封草稿都照你的調性走 💅
你有沒有發現，每次用 Copilot 幫你起草信件，語氣都不太一樣？這封太正式、那封又太隨便，每次都得回頭手動改成「你的味道」，久了比自己寫還累。

其實新版 Outlook 的 草稿指示（Draft instructions）可以記住你的偏好語氣 —— 你只要設定一次（專業、輕鬆、友善，或你自己描述的風格），之後每一次起草，Copilot 都會自動照你的調性寫，不用每封重講。

怎麼設：新版 Outlook → 設定（Settings）→ Copilot → 草稿指示（Draft instructions）→ 自訂指示（Custom Instruction），填入你的偏好風格，按 儲存。從此每封草稿都「很你」—— 精緻、一致、不用每次重調。
  📋 提示詞範本（繁體中文版）
💅 設定你的專屬撰寫風格（填進新版 Outlook → Copilot → 草稿指示 → 自訂指示）

請依照以下風格幫我起草所有郵件：
・語氣：專業但親切，不要過度生硬或太隨便
・長度：簡潔，重點先講，必要時用條列
・開頭/結尾：用「Hi [名字]」開頭，以「Best regards, [我的名字]」結尾
・語言：繁體中文為主，專有名詞保留英文
・避免：冗長鋪陳、過多形容詞、AI 腔的客套話

（依你的實際偏好調整上面每一行）
  📋 Prompt Template (English)
💅 Set your signature email style (paste into the new Outlook → Copilot → Draft instructions → Custom Instruction)

Draft all my emails in the following style:
・Tone: professional but warm—not stiff, not too casual
・Length: concise, lead with the point, use bullets when helpful
・Opening/closing: start with "Hi [name]", end with "Best regards, [my name]"
・Language: English, plain and direct
・Avoid: long wind-ups, excessive adjectives, AI-sounding filler

(Adjust every line above to your real preferences)
💡 使用小撇步
1. 📍 設定路徑 — 新版 Outlook → Settings → Copilot → 草稿指示 → 自訂指示，填好按 Save。
2. ✍️ 寫具體一點 — 與其寫「專業」，不如寫「親切但不囉嗦，重點先講，結尾用 Best regards」，越具體越貼合你。
3. 🔧 隨工作情境微調 — 對外客戶、對內團隊語氣可能不同；需要時臨時在單封草稿補一句覆寫，不必動全域設定。
