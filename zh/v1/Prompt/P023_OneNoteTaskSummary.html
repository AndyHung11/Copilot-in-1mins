📋 OneNote 待辦事項一覽表
OneNote Outstanding Task Summary
✨ OneNote 筆記待辦全掌握
OneNote 的筆記越記越多，卻找不到自己還有哪些事沒做完？這個 Prompt 讓 Copilot 掃描整個 OneNote 章節，把所有未完成的行動項目彙整成一張清楚的表格，包含說明和狀態。一秒掌握所有待辦，從此告別遺漏的焦慮 ✅
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 OneNote 即可使用
請以表格方式摘要本章節中所有我尚未完成的行動項目。為每個行動項目加上說明和狀態。
  📋 Prompt Template (English)
Summarize in a table all the actions I have outstanding across this section. Add a description and a status for each action.
💡 使用小撇步
1. 確保你在正確的 OneNote 章節（Section）中使用此 Prompt，Copilot 會掃描目前所在章節的所有頁面。
2. 如果想限縮到特定主題，可以加上「只看與專案 X 相關的行動項目」。
3. 產出表格後，可以繼續問 Copilot：「哪個行動項目最急迫？請幫我排序優先序。」
4. 建議每週一早上執行一次，讓自己清楚知道這週要處理的重點任務有哪些。
