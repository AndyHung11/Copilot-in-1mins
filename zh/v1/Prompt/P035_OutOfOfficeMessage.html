🏖️ 自動休假回覆信設定
Set Up Your Out-of-Office Auto Reply
✨ 休假前 1 分鐘設好自動回覆
放假前最後一件事，就是設好自動回覆 — 但你知道嗎，Copilot 可以幫你草擬這封信，還能直接排程到你指定的日期！只要告訴它假期時間、代理人和想要的語氣，一切搞定。讓你出發前少一件事要操心 🏖️
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Outlook 即可使用
我將從 [日期] 開始休假至 [日期]。請草擬一則簡短訊息，讓我可以設為自動回覆。請務必列出我的主管以及 [別名/代理人] 作為代理聯絡人。語氣請使用 [專業、熱情…]。
  📋 Prompt Template (English)
I'm going to be out of the office from [date] until [date]. Draft a short message I can set as an automatic reply to emails while I'm out. Be sure to include my manager and [alias(es)] as backup contacts. The tone should be [professional, enthusiastic, ...].
💡 使用小撇步
1. 在 Outlook 中使用，Copilot 可以幫你直接設定自動回覆，不需要手動複製貼上。
2. 把 [日期]、[別名/代理人] 和 [語氣] 都填上具體資訊，產出的信才會準確好用。
3. 如果想要更個人化，可以加上：「請加上一句感謝對方耐心等候的話。」
4. 出發前記得測試一次，傳一封信給自己確認自動回覆有正確觸發。
