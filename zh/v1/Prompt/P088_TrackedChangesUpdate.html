讓 Copilot 幫你更新 Word 文件，還保留格式、全程追蹤修訂讓你逐一審
✍️ 每週一個立即可用的 Copilot Prompt 範例！
從文件撰寫、會議摘要、創意發想，到資料分析，把這些提示詞放進你的 Copilot 試試看。
💡 Tips：提示詞是給 Copilot 的「指令」 — 越清楚、越具體，產出就越貼近你要的結果。歡迎依需求調整、加上你的情境，並 分享給同事一起共學！
本 期 PROMPT
📝 用追蹤修訂更新 Word 文件
Update a Word Doc with Tracked Changes
✨ 從會議、信件、檔案抓資料更新文件，紅線標記、改了什麼一清二楚 📝
你手上有一份要更新的 Word 文件——可能是一份提案、週報或範本——需要把最新的會議結論、信件往來或檔案數據填進去。手動複製貼上不只慢，還很容易把原本精心排好的格式、編號、樣式全弄亂。

這個 Prompt 讓 Copilot 直接在 Word 裡，用其他文件或來源的相關資訊更新、調整或填寫你目前開啟的文件，而且完整保留既有格式與結構。最關鍵的是：所有更新都以追蹤修訂（紅線）呈現，你可以逐一檢視每個調整，接受或拒絕都由你決定。

怎麼用：在 Word 裡開啟要更新的文件，叫出 Copilot，貼上下面的 Prompt（把方括號換成你的來源），它就會把更新當成 redline 套上去。

不用再擔心「AI 會不會把我的格式改爛」——改什麼你看得到，主控權在你手上 ✨。
  📋 提示詞範本（繁體中文版）
📝 用追蹤修訂更新 Word 文件（在 Word 內叫出 Copilot 貼上）

請使用來自 〔會議、電子郵件、檔案、筆記〕 的相關資訊，更新目前開啟的 Word 文件。保留文件既有的格式與結構，並將所有修改以追蹤修訂（紅線）方式標記，讓我可以逐一檢視後再決定接受。
  📋 Prompt Template (English)
📝 Update a Word doc with tracked changes (bring up Copilot inside Word and paste)

Update the open Word document using the relevant information from [meetings, emails, files, notes]. Preserve the document's existing formatting and structure, and track all edits as redlined changes so I can review them before accepting.
💡 使用小撇步
1. 🔓 先開啟 WorkIQ — 要從你的會議、信件、檔案抓資料，務必在 Copilot 開啟 WorkIQ（前稱 Work 模式），否則抓不到組織內容。
2. 📂 指明具體來源 — 把方括號換成明確的來源（如「上週四的專案會議記錄」「與客戶的報價信」），Copilot 才知道去哪找、填得準。
3. 👀 用「檢閱」逐一審 — 改完到 Word 的「檢閱」索引標籤逐筆看追蹤修訂，確認無誤再「全部接受」，有疑慮的單獨拒絕。
