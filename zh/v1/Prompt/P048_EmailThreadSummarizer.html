Email Thread Summary & Action Items
✨ 郵件串再長都不怕，重點和行動項目一次整理好
一封郵件往來了 20 封，你需要讓新加入的同事快速了解狀況……但從頭讀到尾要 15 分鐘，手動整理重點又要再花 10 分鐘。

這個 Prompt 在 Outlook 中執行，Copilot 會自動讀取整個郵件串，整理成 詳細的活動清單，並在旁邊另外產出一份 獨立的行動項目條列清單，整個過程不到一分鐘。

整理好的摘要以 淺顯易懂的語言 呈現，讓同事馬上能夠接手，不需要閱讀原始的漫長郵件串。無論是 跨部門協作追蹤、客戶溝通整理，還是 請假後補進度，這個 Prompt 都能大幅降低溝通成本！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat 即可使用
請摘要這封郵件串的完整活動內容並以清單列出，然後另外產生一份行動項目的獨立條列清單，可直接傳送給我的同事。請使用淺顯易懂的語言，讓同事能快速進入狀況。
  📋 Prompt Template (English)
Create a summary of this email thread with detailed activities in a list and then create a separate Bullet list of actions items to be sent to my colleagues. Please use language that is easily to understand so my colleagues can get us up to speed quickly.
💡 三個情境玩法
1. 🤝 跨部門協作郵件串
快速掌握多方討論重點，確保各方行動項目一致，避免因資訊落差造成延誤。
2. 👥 客戶溝通追蹤整理
整理客戶往來郵件的承諾與待辦事項，在下次會議前做好萬全準備。
3. 🏖️ 請假後補進度
快速了解錯過的郵件討論，馬上跟上節奏，不讓團隊因你的缺席而停頓。
