✨ 簡報一鍵生講義，學員帶走的不只是截圖
每次辦完工作坊，學員的回饋幾乎都是「投影片能分享嗎？」……你給了 PowerPoint，但沒有脈絡說明的簡報，學員回去根本看不懂。

這個 Prompt 在 Word 中執行，並連結你的 PowerPoint 簡報，Copilot 會自動產生一份完整的工作坊講義：先是簡短的 內容摘要，接著是結構化的 關鍵要素表格（要素 / 簡短說明），最後附上針對內容提出的 延伸挑戰問題與解答。

無論是 技術工作坊、主管簡報後分享，還是 教育訓練課程，一份好的講義讓學習效果延伸到活動結束之後！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat 即可使用
請製作一份講義，先以簡短摘要介紹 /[簡報或文件] 的內容，接著建立一個包含關鍵要素的表格，表格欄位為「關鍵要素」與「簡短說明」。表格之後，根據 / 中的內容列出延伸性與挑戰性問題及其解答。
  📋 Prompt Template (English)
Create a handout, starting with a short summary of the content, followed by a table with the key elements of the content. The table should have the columns key elements, short description. Below the table, present further and critical questions and their answers regarding the content based on /.
💡 三個情境玩法
1. 🔬 技術工作坊
學員帶走完整技術摘要與深度問答，強化學習效果，減少事後 Q&A 追問。
2. 📤 主管簡報後分享
讓無法出席的同事也能快速理解報告重點，人人都能跟上進度。
3. 📚 教育訓練課程
自動產生課後複習講義，節省講師準備時間，讓學員有充分的自學資料。
