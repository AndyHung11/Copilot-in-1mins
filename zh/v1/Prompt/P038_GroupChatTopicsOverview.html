Group Chat Topics Overview
✨ 7 天訊息一表掌握，再也不漏接重要討論
每次一打開 Teams 群組聊天，就看到幾百則訊息在等你……從截圖、表情包到重要公告全混在一起，要找關鍵討論真的要命。

這個 Prompt 在 Teams 的 Copilot 功能 中執行，直接幫你把過去 7 天的群聊訊息，按照討論主題整理成一張張獨立表格：每個主題有 發起人、開始時間、TLDR 結論、詳細摘要，以及是否「待回覆」的狀態標記。

無論是 週例會前複習、請假後趕進度，還是 跨部門協作追蹤，這個 Prompt 讓你 60 秒以內掌握 7 天所有重點！
  📋 提示詞範本（繁體中文版）
※ 直接複製整段，貼到 Copilot Chat 即可使用
建立此聊天室過去 7 天訊息的總覽，按從新到舊排序。請完整遵照以下格式說明。將每個主題各自整理成一個獨立的表格，每個表格只有一個儲存格。在這個儲存格中加入各主題的定義內容，各主題之間以兩個換行分隔。在各主題摘要的儲存格內，依序加入：兩個換行後以粗體標示主題標題（簡短詞語）；兩個換行後以粗體顯示「Topic started by:」並寫出發起人（一般字型）；兩個換行後以粗體顯示「Started on:」並寫出星期縮寫與日期（DD.MM.，一般字型）；兩個換行後以粗體顯示「TLDR:」並以一般字型寫出主題結論；兩個換行後以粗體顯示「Content of the discussion:」並以一般字型詳細摘要對話內容；兩個換行後以粗體顯示「Topic waiting for a reply:」並以是／否（一般字型）說明是否有未回覆的問題。最後加兩個換行。
  📋 Prompt Template (English)
Create an overview of all the messages in this chat of the last 7 days sorted by newest to oldest. Follow the formatting instructions exactly. Summarize every topic in a separate table, with only one cell each. Add each topic with the defined contents to this one cell. Add two line breaks between each topic and its table. In the summary of the topics, in the individual cells, add the following contents. Start with two line breaks, then create the heading of the topic in bold. Create a good term that reflects the topic briefly and concisely. Then add two line breaks. Continue in a new line with the title "Topic started by:", write this in bold. Name the person who started the topic. Then add two line breaks. Continue in a new line with "Started on:" write this in bold. Name the weekday (abbreviated) and the date (in the format DD.MM.) in normal font. Then add two line breaks. Continue in a new line with the point "TLDR:" in bold. Create a summarizing conclusion for the topic, in normal font. Then add two line breaks. Continue in a new line with the point "Content of the discussion:" in bold. Summarize the content of the conversation in detail, all in normal font. Then add two line breaks. Continue in a new line with the point: "Topic waiting for a reply:" in bold. Add a yes or no here, as an indication of whether a question or topic is still unanswered, in normal font. Add two line breaks at the end.
💡 三個情境玩法
1. 📅 週例會前快速複習
開會前 3 分鐘用這個 Prompt 看完群聊重點，準備充分、不踩坑。
2. 🏖️ 請假回來趕進度
假期結束後迅速補回所有錯過的討論，馬上接手工作不落後。
3. 🏢 跨部門協作群組
掌握多方討論進度，確認沒有遺漏你負責的待辦事項或決策請求。
