打造你的個人化 Copilot 指令
🎯 個人化 · Personalization
每天都在 Outlook 寫信、在 Teams 跟同事打字，你有沒有想過 — 你的「溝通風格」其實已經悄悄被資料記錄下來了？

這個 Prompt 會請 Copilot：
🔍 分析你最近的 Outlook 信件與 Teams 對話，偵測你的語氣、用字模式與結構。
🧬 推導出你的「溝通基因檔案」 — 你的核心價值、強項、弱點與技能（不看單一訊息，而是找一致的行為模式）。
📦 產出兩份個人化指令：
    1️⃣ Copilot Chat 指令（角色定位、語氣、回覆結構、預設值）
    2️⃣ Outlook 撰寫指令（主旨、開場/結尾、語氣/長度、結構、Do / Don't）

一次產出，讓 Copilot 變成「最懂你」的版本，每封信、每段對話都更貼近你本來的樣子！
📝 提示詞
※ 完整複製貼上即可使用
請幫我為 Copilot 的「個人化（Personalization）」功能產出一組指令。請先讀過我最近的 Outlook 信件與 Teams 聊天訊息，從中找出我的核心價值、強項、弱點與技能，做一次完整的「溝通風格」分析。根據分析結果，請分別產出兩份指令：第一份給 Copilot Chat 使用、第二份給 Outlook 使用，後者要能直接用在草擬 email 上。
英文版
I want you to help me create a set of instructions for the personalization feature in Copilot. Read through my recent emails and chats from Teams and provide a comprehensive analysis of my communication style by identifying my core values, strengths, weaknesses and skills. Based on the analysis, generate instructions that I can use in Copilot Chat and a second set for Outlook so that I can use it for e-mail drafting
🚀 適用情境： Microsoft 365 Copilot Chat（Work 模式）
💭 小提醒：產出後可貼到 Copilot → 個人化設定 → 自訂指令，並另存到 Outlook 寫信偏好中。
🔑 需要 M365 Copilot 授權
