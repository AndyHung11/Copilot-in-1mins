本 期 PROMPT
🪞 打造你的個人化指令
Personalize Your Comms Style
✨ 讓 Copilot 寫出「像你會寫的」內容
有沒有發現？Copilot 預設寫出來的東西雖然 OK，但總覺得「不太像自己的口吻」。其實它不是不會寫，而是還不認識你！

只要把這份 個人化指令 一次設定好，告訴它你的職稱、團隊、常往來對象、寫作風格與愛用的工具，之後每一封郵件、每一份會議摘要、每一段 Teams 訊息，Copilot 都會用 更貼近你風格 的方式產出。

一次設定，長期受用 — 從此告別「再幫我重寫一次，口吻像我一點」的迴圈！
  📋 提示詞範本（繁體中文版）
※ 把 {方括號} 內的內容換成自己的資訊，再貼到 Copilot 對話框
請依照以下我的個人資訊，調整你協助我撰寫郵件、回覆訊息、產生摘要時的風格與內容： 【關於我】 - 姓名與職稱：{你的姓名} / {你的職稱} - 部門 / 團隊：{部門名稱} - 主要工作職責：{2–3 句描述你日常負責的事務} 【常往來對象】 - 直屬主管：{主管姓名與職稱} - 重要協作夥伴：{部門 / 角色名稱} - 常聯繫的客戶或外部單位：{舉例} 【慣用語言與風格】 - 主要工作語言：{繁體中文 / 英文 / 雙語} - 寫作風格：{例：簡潔、正式、帶一點親切感} - 通常使用的稱呼：{Hi {Name} / 您好 / Dear {Name}} 【常用工具與資料來源】 - 主要使用的 Microsoft 365 工具：{Outlook / Teams / SharePoint / OneDrive / Loop} - 常引用的資料夾或網站：{舉例} 【特別注意事項】 - {例：對外信件請保持正式且加上完整敬語} - {例：對主管的更新請以條列式呈現重點} - {例：避免使用過於口語的縮寫} 請記住以上資訊，未來協助我撰寫任何內容時，主動套用這些風格設定。
  📋 Prompt Template (English)
Please use the following personal context to tailor how you help me draft emails, reply to messages, and create summaries: [About Me] - Name & Title: {Your Name} / {Your Title} - Department / Team: {Department Name} - Key Responsibilities: {2–3 sentences about your day-to-day work} [Frequent Contacts] - Direct Manager: {Manager's name & title} - Key Collaborators: {Departments / Roles} - Frequent Customers or External Parties: {Examples} [Preferred Language & Style] - Primary Working Language: {English / Traditional Chinese / Bilingual} - Writing Style: {e.g., concise, formal, slightly friendly} - Default Greeting: {Hi {Name} / Dear {Name}} [Common Tools & References] - Main Microsoft 365 Tools: {Outlook / Teams / SharePoint / OneDrive / Loop} - Frequently Referenced Folders or Sites: {Examples} [Special Notes] - {e.g., External emails must remain formal and include full sign-off} - {e.g., Updates to my manager should be in bullet format with key takeaways} - {e.g., Avoid casual abbreviations} Please remember the above and proactively apply these style preferences whenever you help me write content in the future.
💡 使用小撇步
1. 第一次貼上後，可以請 Copilot：「請複述一下你記住的我的個人風格」確認是否正確收到。
2. 之後在新對話開頭簡短提一句：「請依照我之前提供的個人風格設定協助我」，效果最佳。
3. 風格會隨時間調整，建議 每季回頭檢視一次 並更新內容。
