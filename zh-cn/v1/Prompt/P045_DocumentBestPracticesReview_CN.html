💬 文档最佳实践差距分析
Document Best Practices Review
✨ 文档交出前，先让 Copilot 帮你找漏洞
文档交出去前，总担心漏了什么关键段落，但又没时间逐项对照标准？

这个 Prompt 让 Copilot 把你的文档对照最佳实践指南，自动找出尚未涵盖的要点、列出每一个可以再强化的地方，以清单呈现方便逐项处理。在 Copilot Chat 中同时上传主文档与指南，一次完成对照。

交件前先抓漏洞，让每份文档都更有把握！
  📋 提示词模板（简体中文版）
请查看我的 [document.docx]，并对照此处列出的最佳实践 [Best-Practices.pdf] 进行分析。列出我们尚未涵盖的内容，以及还可以进一步改进的地方。
  📋 Prompt Template (English)
Review my [document.docx] and analyze it against the best practices listed here [Best-Practices.pdf]. Make a list of the things that we do not yet cover and the things where we can still improve.
💡 使用技巧
1. 📐 产品规格书审查
确认技术文档完整符合公司规范与业界标准，交出前有充分把握。
2. 📤 提案文档升级
在送出前确认提案涵盖所有关键要素，提高被采纳的几率。
3. 🔄 流程文档稽核
对照质量标准找出流程文档的遗漏与改善点，确保流程文档始终保持最新。
