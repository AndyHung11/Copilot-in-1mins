本期 PROMPT
🧩 问题解决脑力激荡伙伴
Problem Solving Buddy
✨ 迭代提问＋量身解法，突破思考盲点
碰到棘手问题，自己想不到解法，问别人又得不到有深度的建议？这个 Prompt 把 Copilot 变成你的问题解决伙伴！它先了解你的问题，再问你三个最关键的问题来深挖，然后提出量身打造的解法 — 而且每轮都追加三个新问题继续迭代，帮你从多个角度突破盲点。问题越难，越要有好伙伴 🧩
  📋 提示词模板（简体中文版）
我需要你成为我的问题解决脑力激荡伙伴。请先问我有什么问题或机会。然后提出 3 个最相关的问题，帮助我找到最佳解决方案。根据我的回答，提出解决我挑战的方案，并每次都附加 3 个新问题来持续迭代解法。
  📋 Prompt Template (English)
I need you to become my problem-solving brainstorm buddy. Start by asking me about my problem or opportunity. Then ask me the 3 most relevant questions to help me find the best solution. Based on my answers, propose a solution to my challenge and always add 3 new questions to iterate on the solution.
💡 使用技巧
1. 一开始只要说出问题的大方向就好，让 Copilot 的问题引导你说清楚细节。
2. 回答 Copilot 的问题时，尽量具体，越多真实背景信息，解法越量身订制。
3. 觉得某个解法方向有趣，可以说：「请针对这个方向再深入探讨」，继续迭代。
4. 这个 Prompt 适合任何类型的问题：工作挑战、策略规划、人际关系，甚至创意发想都很好用。
