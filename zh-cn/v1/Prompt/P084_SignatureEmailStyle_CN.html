💅 帮每封信设置专属撰写风格
Set Your Signature Email Style
✨ 设置一次撰写风格，之后每封草稿都照你的调性走 💅
你有没有发现，每次用 Copilot 帮你起草信件，语气都不太一样？这封太正式、那封又太随便，每次都得回头手动改成「你的味道」，久了比自己写还累。

其实新版 Outlook 的 草稿指示（Draft instructions）可以记住你的偏好语气 —— 你只要设置一次（专业、轻松、友善，或你自己描述的风格），之后每一次起草，Copilot 都会自动照你的调性写，不用每封重讲。

怎么设：新版 Outlook → 设置（Settings）→ Copilot → 草稿指示（Draft instructions）→ 自订指示（Custom Instruction），填入你的偏好风格，按 保存。从此每封草稿都「很你」—— 精致、一致、不用每次重调。
  📋 提示词模板（简体中文版）
💅 设置你的专属撰写风格（填进新版 Outlook → Copilot → 草稿指示 → 自订指示）

请依照以下风格帮我起草所有邮件：
・语气：专业但亲切，不要过度生硬或太随便
・长度：简洁，重点先讲，必要时用条列
・开头/结尾：用「Hi [名字]」开头，以「Best regards, [我的名字]」结尾
・语言：繁体中文为主，专有名词保留英文
・避免：冗长铺陈、过多形容词、AI 腔的客套话

（依你的实际偏好调整上面每一行）
  📋 Prompt Template (English)
💅 Set your signature email style (paste into the new Outlook → Copilot → Draft instructions → Custom Instruction)

Draft all my emails in the following style:
・Tone: professional but warm—not stiff, not too casual
・Length: concise, lead with the point, use bullets when helpful
・Opening/closing: start with "Hi [name]", end with "Best regards, [my name]"
・Language: English, plain and direct
・Avoid: long wind-ups, excessive adjectives, AI-sounding filler

(Adjust every line above to your real preferences)
💡 使用技巧
1. 📍 设置路径 — 新版 Outlook → Settings → Copilot → 草稿指示 → 自订指示，填好按 Save。
2. ✍️ 写具体一点 — 与其写「专业」，不如写「亲切但不啰嗦，重点先讲，结尾用 Best regards」，越具体越贴合你。
3. 🔧 随工作场景微调 — 对外客户、对内团队语气可能不同；需要时临时在单封草稿补一句覆写，不必动全域设置。
