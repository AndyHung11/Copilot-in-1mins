♿ 无障碍文档审查助理
Accessibility Review
✨ 一键审查文档无障碍问题，确保每个人都能访问你的内容
你的文档，确定每个人都读得到吗？屏幕阅读器、色彩对比、图片替代文本——这些细节很容易被忽略。

把文档粘贴或上传，这个 Prompt 就会请 Copilot 依无障碍标准全面审查，找出每一个问题，并针对屏幕阅读器兼容性、色彩对比度、替代文本等面向给出具体改善建议，确保符合 WCAG 等指引。

让你的内容对所有人——包括残障用户——都一样友善！
  📋 提示词模板（简体中文版）
请审查此文档，确保其符合无障碍标准。识别任何无障碍问题并提供改善建议。确保文档完全无障碍且符合相关无障碍指引。此文档面向多元受众，包括残障用户。它应能供屏幕阅读器使用、具备适当的色彩对比度，并为图片提供替代文本。请确保生成内容详尽，提供详细的改善建议。
  📋 Prompt Template (English)
Review this document to ensure it meets accessibility standards. Identify any accessibility issues and provide recommendations for improvement. Ensure that the document is fully accessible and compliant with relevant accessibility guidelines. The document is intended for a diverse audience, including individuals with disabilities. It should be accessible to people using screen readers, have appropriate color contrast, and include alternative text for images. Please make sure the output is exhaustive and provides detailed recommendations.
💡 使用技巧
1
📄 先给 Copilot 完整内容
粘帖子档内容或上传文件后，再贴这段提示词，让 Copilot 有完整上下文可以审查。
2
🧩 用三栏整理改善点
要求 Copilot 用「问题 / 影响 / 建议修正」输出，方便你逐项修改与追踪。
3
✅ 修完再跑一次复查
完成修正后再运行一次审查，确认标题、替代文本、链接、表格与色彩对比都符合需求。
