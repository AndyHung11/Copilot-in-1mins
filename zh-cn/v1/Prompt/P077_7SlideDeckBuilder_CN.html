本期 PROMPT
🎯 七页演示文稿快速产生器
7-Slide Deck Builder
✨ 一段文本，几分钟变成可上场的 7 页演示文稿
客户下午丢给你一段 Brief，明天就要 Pitch。没有模板、没有 add-in、没有设计师——你只有 Copilot 和那段文本。

这个 Prompt 会请 Copilot 把任何杂乱的 Brief 变成一份结构清楚的 7 页演示文稿：标题、问题／脉络、解决方案三支柱、价值与成果、交付计划、案例、下一步与 Call to Action——而且每一页都附上 2–4 个 bullet 的演讲备注。

缺信息？Copilot 会直接插上 [Info needed: …] 的占位符，提醒你回头补。提案、进度更新、内部 Pitch 都适用。
  📋 提示词模板（简体中文版）
根据下方的 Brief，产生一份精简的 7 页演示文稿。输入：仅以粘贴的文本作为来源。投视频（共 7 页，顺序固定）：1) 标题 — [项目／提案名称] 2) 问题／脉络 3) 解决方案（最多 3 个支柱）4) 价值与成果（尽量量化）5) 交付计划与里程碑（简单的时间轴 bullet）6) 案例／实证（Case、Pilot 或代理指标）7) 下一步与行动呼吁。要求：每页加上 2–4 个 bullet 的演讲备注；使用清楚、好读的条列（每页最多 5 条）；不需品牌或图片，中性主题即可；若关键细节缺失，请插入占位符 [Info needed: …]。Brief 开始：[此处粘贴 Brief] Brief 结束。
  📋 Prompt Template (English)
Create a concise 7-slide deckfrom the brief below. Inputs: Use only the pasted text as the source. Slides (exactly): Title — [PROJECT/PROPOSAL NAME] Problem/Context Proposed Solution (3 pillars max) Value & Outcomes (quantify if possible) Delivery Plan & Milestones (timeline in simple bullets) Evidence/Example (case, pilot, or proxy) Next Steps & Call to Action Requirements: Add concise speaker notes to each slide (2–4 bullets). Use clear, readable bullets (max 5 per slide). No branding or images; neutral theme is fine. If crucial details are missing, insert a placeholder bullet [Info needed: …]. Brief Starts: [PASTE BRIEF HERE] Brief Ends.
💡 使用技巧
1. 在 Copilot Chat 使用 PowerPoint agent，粘贴完整 Prompt 即可生成投视频内容（免许可）。
2. 把 [项目／提案名称] 换成你实际的案名，并把 Brief 完整贴入。
3. 看到 [Info needed: …] 就回头补数据，Copilot 会自动接着生成完整内容。
