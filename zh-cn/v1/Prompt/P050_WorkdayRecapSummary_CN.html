✨ 一个表格搞定邮件和 Teams，今天的事今天结清
一天结束，邮件和 Teams 消息散落各处，到底有哪些事还没结？

这个 Prompt 让 Copilot 把你上一个工作日的邮件和 Teams 消息集成成一张表格（类型／主题／摘要／行动项目／后续行动），还会建议每个行动的下一步，你被直接点名的主题自动加粗。

一个表格，把今天的事今天结清！
  📋 提示词模板（简体中文版）
请摘要我上一个工作日的邮件和 Teams 消息，并在独立字段中列出行动项目。如果可以的话，请在另一个独立字段中建议后续行动。表格格式如下：类型（邮件/Teams）| 主题 | 摘要 | 行动项目 | 后续行动。如果我被直接点名，请将该主题的字体加粗。
  📋 Prompt Template (English)
Summarize my emails and Teams messages from the last workday. List action items in a separate column. If possible, suggest follow-up actions in a separate column. The table should look as follows: Type (Mail/Teams) | Topic | Summary | Action Item | Follow-up Action. If I was mentioned directly, please make the font of the topic bold.
💡 使用技巧
1. 🌙 每日下班收尾仪式
快速确认今日工作状态，有条不紊地结束一天，带着清晰的心情离开办公室。
2. ☕ 周一早晨补课
周末后回来时，迅速掌握周五下班后的消息动态，快速进入工作节奏。
3. 🌏 跨时区协同追踪
确认非工作时间收到的跨国消息中有哪些待办任务，让协同不因时差而延迟。
