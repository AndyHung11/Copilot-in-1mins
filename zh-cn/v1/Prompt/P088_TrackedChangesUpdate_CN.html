💡 使用技巧：提示词就是给 Copilot 的“工作指令”——越清晰、越具体，输出越贴近你的目标。可结合自己的业务场景调整，并 分享给同事共同学习。
本期 PROMPT
📝 用追踪修订更新 Word 文档
Update a Word Doc with Tracked Changes
✨ 从会议、信件、文件抓数据更新文档，红线标记、改了什么一清二楚 📝
你手上有一份要更新的 Word 文档——可能是一份提案、周报或范本——需要把最新的会议结论、信件往来或文件数据填进去。手动拷贝粘贴不只慢，还很容易把原本精心排好的格式、编号、样式全弄乱。

这个 Prompt 让 Copilot 直接在 Word 里，用其他文档或来源的相关信息更新、调整或填写你目前打开的文档，而且完整保留既有格式与结构。最关键的是：所有更新都以追踪修订（红线）呈现，你可以逐一查看每个调整，接受或拒绝都由你决定。

怎么用：在 Word 里打开要更新的文档，叫出 Copilot，粘贴下面的 Prompt（把方括号换成你的来源），它就会把更新当成 redline 套上去。

不用再担心「AI 会不会把我的格式改烂」——改什么你看得到，主控权在你手上 ✨。
  📋 提示词模板（简体中文版）
📝 用追踪修订更新 Word 文档（在 Word 内叫出 Copilot 粘贴）

请使用来自 〔会议、电子邮件、文件、笔记〕 的相关信息，更新目前打开的 Word 文档。保留文档既有的格式与结构，并将所有修改以追踪修订（红线）方式标记，让我可以逐一查看后再决定接受。
  📋 Prompt Template (English)
📝 Update a Word doc with tracked changes (bring up Copilot inside Word and paste)

Update the open Word document using the relevant information from [meetings, emails, files, notes]. Preserve the document's existing formatting and structure, and track all edits as redlined changes so I can review them before accepting.
💡 使用技巧
1. 🔓 先打开 WorkIQ — 要从你的会议、信件、文件抓数据，务必在 Copilot 打开 WorkIQ（前称 Work 模式），否则抓不到组织内容。
2. 📂 指明具体来源 — 把方括号换成明确的来源（如「上周四的项目会议记录」「与客户的报价信」），Copilot 才知道去哪找、填得准。
3. 👀 用「检阅」逐一审 — 改完到 Word 的「检阅」索引标签逐笔看追踪修订，确认无误再「全部接受」，有疑虑的单独拒绝。
🔑 需要 M365 Copilot 许可
