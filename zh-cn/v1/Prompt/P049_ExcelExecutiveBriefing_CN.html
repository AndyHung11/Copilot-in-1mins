💬 Excel 一页式负责人决策演示文稿
Excel Executive Briefing
✨ 把密密麻麻的 Excel 变成负责人一眼看懂的决策工具
负责人没空看密密麻麻的电子表格，只想要一页就能决策的重点，你要怎么浓缩？

这个 Prompt 让 Copilot 把 Excel 档整理成一页式负责人决策演示文稿：自动提取关键 KPI、标出趋势与警示红旗，还挑选最合适的图表（长条图或圆饼图）可视化，集成成适合投影或邮件的单页摘要。

把一堆数字，变成负责人一眼就懂的决策工具！
  📋 提示词模板（简体中文版）
请将此 Excel 档整理成一页式负责人决策演示文稿，包含 KPI、趋势与任何警示红旗。请提供一个合适的数据集可视化图表（例如长条图或圆饼图）。此图表应帮助用户快速发现相关性与趋势，进一步强化一页演示文稿的效果。
  📋 Prompt Template (English)
Summarize this Excel file into a one-page executive briefing. Include KPIs, trends, and any red flags. Give me a suitable visual representation of the dataset in a graph of your choice - for example, a bar or pie chart. This should help the user quickly notice correlations and trends, adding more complement to the one-pager's intent.
💡 使用技巧
1. 💼 季度业绩报告
把密密麻麻的财务数字转化成一张负责人可以带进董事会的精华摘要。
2. 📣 销售漏斗分析
从转换率数字中自动找出瓶颈并可视化呈现，快速定位需要强化的环节。
3. 📡 项目状态追踪
将多个指标集成成一目了然的项目健康状态图，让所有人共享同一份数据视角。
