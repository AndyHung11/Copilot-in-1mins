本期 PROMPT
📥 信件网域摘要
Email Domain Inbox Summary
✨ 一个 Prompt 看完所有客户来信＋谁已回
客户下午突然问：「上周那个案子我们同事有没有回？」——你开始翻邮件、跨过 8 个对话串、查同事副本、确认谁回了什么。10 分钟过去，还没给答案。

这个 Prompt 直接在 Outlook Copilot 用：指定一个网域（例如 @customer.com）与时间范围（例如过去 4 周），就回一张表——谁寄的、何时寄的、需要什么行动、我方有没有人回过。

对于客户经理、PM、业务、Account 角色超实用：每天开工前跑一次，马上知道哪些 thread 还没人接、哪些已经处理掉，把「漏接客户来信」这个风险直接消掉。
  📋 提示词模板（简体中文版）
创建一张表格，列出我在过去 X 周内从 DOMAINNAME（例如 @customer.com）收到的所有 email。请包含一栏显示这封信需要哪些行动，以及另一栏显示我公司是否有人已经回复。
  📋 Prompt Template (English)
Create a tableshowing all emails I received from DOMAINNAME (e.g., @customer.com) in the last X weeks. Include a column that shows any action required from the message and another column to show if anyone at my company has responded yet.
💡 使用技巧
1. 在 Outlook 的 Copilot 直接粘贴（行动设备或桌面版皆可）。
2. 把 DOMAINNAME 换成实际网域（例如 @chunghwa.com.tw），X 周换成你要的范围（1、2、4、8…）。
3. 看到「尚无人回复」的列就先处理；看到「已回复」的列再确认回得对不对、要不要追加。
