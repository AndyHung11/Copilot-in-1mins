💬 多文档 FAQ 快速产生器
Multi-Document FAQ Generator
✨ 把一叠文档浓缩成一份 FAQ，让每个人都能快速上手
你有一份详细的产品规格演示文稿和一份设计文档……但新来的工程师说他不知道从哪里读起，客户说找不到他要的信息……每次都要手动回答同样的问题，真的很累。

这个 Prompt 在 Word 中运行，你可以同时链接 多份文档（演示文稿、报告、会议记录等），Copilot 会自动集成内容，输出一份包含 清楚标题、条列点，并附上完整解答的 FAQ 文档。

输出支持 Markdown 格式，可直接贴到 Teams 聊天、SharePoint 或 Wiki 页面，让每个人都能自助找到答案，从此告别重复回答同样问题！
  📋 提示词模板（简体中文版）
请依据 [presentation.pptx] 和 [document.docx] 创建一份 FAQ，包含清楚的标题与条列点以提升可读性。确保文档结构能有效回应常见问题，并为目标读者提供完整的解答。考虑使用 Markdown 格式化文档，方便分享与分发。
  📋 Prompt Template (English)
Create an FAQ based on [presentation.pptx] and [document.docx] that includes clear headings and bullet points to enhance readability. Ensure that the document is structured in a way that addresses common questions effectively and provides comprehensive answers for the intended audience. Consider using Markdown to format the document for easy sharing and distribution.
💡 使用技巧
1. 🚀 新产品上市文档包
把产品规格和演示文稿浓缩成 FAQ，加速客户和内部人员的理解，减少反复问答。
2. 🔄 项目交接数据集成
为接手的同事产生一份快速上手的问答文档，降低交接成本。
3. 📚 内部知识库建设
将各式文档转化为可搜索的 FAQ 知识库条目，让组织知识得以保存和传承。
