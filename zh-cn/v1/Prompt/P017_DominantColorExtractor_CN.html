🎨 🎨 图片主色调提取工具
Dominant Color Extractor
✨ 上传任何图片，一秒取得 RGB 和 HEX 色码，搞定设计配色
你有没有看到一张很漂亮的图片，想把它的配色用在你的演示文稿或设计上，但根本不知道那些颜色的确切色码？或者你需要确保品牌视觉一致性，但手边没有设计工具？

这个 Prompt 让你把任何图片或截屏上传给 Copilot Chat - Web，立刻取得最主要的8种颜色，包含 RGB 数值和 HEX 色码，以及每个颜色的描述标签（支持无障碍设计）。完全不需要设计工具，也不需要 M365 许可。
  📋 提示词模板（简体中文版）
提取这张图片中的 [8] 个主要颜色。提供每个颜色的 RGB 数值、HEX 色码，以及每个颜色的简短描述（例如用于无障碍设计或设计用途）。
  📋 Prompt Template (English)
Extract the [8] dominant colors from this image. Provide the RGB values, HEX codes, and a short description of each color (e.g., for accessibility or design purposes).
💡 使用技巧
📸 上传图片再运行：把图片或截屏直接上传到 Copilot Chat（Web），再粘贴这个 Prompt。
🔢 调整提取数量：把 [8] 改成 [5] 或 [12]，根据需要提取更少或更多的颜色。
🎯 指定颜色用途：说「这些颜色用于营销素材配色」让描述标签更贴合你的应用场景。
🆓 无需 M365 Copilot 许可，基础版即可使用
