🧠 激活 Copilot Memory
Copilot Memory
✨ 一次教会 Copilot 你是谁、怎么工作
每次跟 Copilot 对话，是不是都要从头交代一次「我是谁、做什么、偏好什么风格」？烦死了对吧。

Copilot Memory 是新功能 — 它会记住你的偏好、工作风格、关键细节，让对话更有效率、更贴近你的场景。只要你说 「Copilot, 我比较偏好…」 或 「请记住…」，Copilot 就会在画面上出现一个小通知，把这条信息存进你的记忆档。

这周的 Prompt 就是 Copilot Memory 的 启动钥匙 — 它会问你三个简单的问题（工作风格、写作风格、角色重点），然后一次把答案存进你的 Copilot Memory。之后你随时可以在设置里 查看、删除、添加 任何一条记忆！
  📋 提示词模板（简体中文版）
帮我启动我的 Copilot Memory。请向我提出三个简单问题：分别关于我的工作风格、写作风格，以及我的职务角色的关键事实。请把我的回答加到我的 Copilot Memory 个人文件中。
  📋 Prompt Template (English)
Help me with getting started on my Copilot Memory. Ask me three simple questions about my work style, writing style and key facts about my role. Add the results to my Copilot Memory profile.
💡 使用技巧
1
🔍 第一次跑完马上验收
问 Copilot：「请复述你刚刚帮我记住的每一条 Memory」确认它有完整收到。
2
➕ 随时加新记忆
日常对话中说「Copilot, 我比较偏好…」或「请记住…」就会看到小通知，直接存进 Memory。
3
🧹 每季回头整理
打开 Copilot 设置 ➜ Memory，删掉过时或不再适用的旧偏好，保持精准。
🆓 无需 M365 Copilot 许可，基础版即可使用
