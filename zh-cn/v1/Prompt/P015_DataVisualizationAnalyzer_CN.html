本期 PROMPT
📈 📈 数据可视化与洞察分析
Data Visualization Analyzer
✨ 上传数据，Analyst Agent 自动找出什么在成长、什么在下滑、为什么
你有一份财务、销售或营销数据，但打开 Excel 后却不知道从哪开始分析？或者你做出了一堆图表，却说不清楚「所以呢？数字背后的故事是什么？」

Analyst Agent 让你上传一个或多个数据文件，它会自动分析、可视化，并找出哪些指标表现好、哪些需要关注，以及是什么在驱动这些变化。你可以把「areas」替换成任何 KPI，例如产品、服务、客户区段或营销活动。
  📋 提示词模板（简体中文版）
分析并可视化这份数据，找出表现良好的 [指标/区域] 和表现不佳的 [指标/区域]。什么因素对这些变化影响最大？另外，如果数据允许，请分析细分市场或地区绩效。
  📋 Prompt Template (English)
Analyze and visualize this data and identify [areas] that are doing well and [areas] that are not doing so well. What is influencing changes the most? Additionally, if data is available, analyze segment or regional performance.
💡 使用技巧
📎 上传数据文件：在 Analyst Agent 中上传 Excel、CSV 或其他数据文件，再粘贴这个 Prompt。
🔄 替换 [指标/区域]：可以换成「产品线」、「销售地区」、「营销活动」等你实际关注的维度。
🔍 深入追问：收到初步分析后说「请针对表现最差的地区展开原因分析」取得更深入的洞察。
