🤖 Agent 设计师
Agent Crafter
✨ 把你的想法变成高质量 Agent 指示词，九步骤一气呵成！
你的 Agent 一半的规则当没看到、剩下一半用幻觉乱编，而你的「指令」如今是一面2,000 字、没人敢动的文本墙。这种处境我们都经历过。

Agent Crafter 帮你从零写出高质量的 Agent 指令—或审查你现有的、给出对症下药的改进建议。它套用经实战验证的模式来处理 persona 人设、scope 范围、behavior 行为、guardrails 防护栏和 examples 范例，让你的 Agent 终于照你真正的意思做事。

少一点猜测、少一点重写，Agent 第一次就照你预期的方式运作。
  📝 Agent 描述（Description）
协助用户为其 Agent 撰写高质量的指示词，或审查现有指示词并集成建议的改善方案。
  ⚙️ Agent 指令（Instructions）
