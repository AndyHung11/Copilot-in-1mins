上周交办你的事散落在 Email、会议逐字稿、Teams 消息、文件注解里，周一一打开电脑根本不知道从哪开始？

这个 Prompt 让 Copilot 把所有「指派给我」「@我」「请你跟进」的事项一次捞出来，整理成「事项 / 指派人 / 截止日 / 状态」四栏的表格。把它设置成 Scheduled Prompt，每周一早上 8:30 自动跑，开工前清单已经躺在信箱里。
🎁 你将获得什么？
📋 一张清单看完所有 Action
事项、指派人、截止日、状态四栏，一目了然。
🔄 跨来源集成
Email + 会议逐字稿 + Teams + 文件 @mention 一次捞完。
⏰ 自动调度跑
设成 Scheduled Prompt，每周一自动输出寄到信箱。
📌 直接接 Planner / To Do
表格输出后直接拷贝到任务管理工具追踪。
🚀 直接复制这段 Prompt（简中版）
请汇总上周通过 电子邮件、会议逐字稿、Teams 消息、以及我被 @ 提及的文件注解 中所有指派给我的待办任务。包含任何别人提到的任务、后续追踪请求或截止日期。请创建一张表格，包含以下字段：
1) 待办任务（Action Item）
2) 由谁指派（Who Assigned It）
3) 请求的截止日（Requested Due Date）
4) 状态（Status）（若有提到）
若没有提到截止日，字段请留空。
📋 Prompt Template (English)
Please compile all action items assigned to me last week from my emails, meeting transcripts, Teams messages, and file comments where I was @-mentioned. Include any tasks, follow-up requests, or deadlines that others raised for me. Create a table with the following columns:
1) Action Item
2) Who Assigned It
3) Requested Due Date
4) Status (if mentioned)
If no due date was mentioned, leave the field blank.
💡 三个典型使用场景
1
⏰ 设成 Scheduled Prompt 每周一跑
用 aka.ms/ScheduledPrompt 排成每周一 8:30，开工前清单已寄到信箱。
2
📌 一键接到 Planner / To Do
输出表格后直接拷贝到 Microsoft To Do 或 Planner，立刻变成可追踪任务。
3
🔍 换时间范围对应不同查看
把「上周」改成「过去 14 天」「本月」「本季」就能对应周、双周、月、季的回顾节奏。
