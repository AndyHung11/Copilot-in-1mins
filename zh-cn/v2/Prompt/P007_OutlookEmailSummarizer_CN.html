Copilot Chat 现在在 Outlook 开放给所有人使用（不需 M365 Copilot 许可）！碰到一条长长的邮件串，不知道从哪读起？

这个 Prompt 帮你用两句话抓出重点、标出哪些任务是指派给你的，还顺手生成一份回复草稿，让你快速确认后就可以送出。从此邮件再长也不怕，处理速度直接翻倍。
🎁 你将获得什么？
✏️ 两句话内核摘要
快速掌握邮件串的关键消息，不必从头逐字阅读。
✅ 我的任务清单
自动标出邮件串中指派给你的所有待办任务。
📝 回复草稿自动生成
根据邮件内容产生草稿，确认后即可送出。
🚀 直接复制这段 Prompt（简中版）
用两句话摘要这封邮件。在下一节中，标示邮件串中指派给我的任何任务。然后为我的回复创建一份草稿。
📋 Prompt Template (English)
Summarize this mail in two sentences. In the next section, highlight any task given for me in the mail thread. Then create a draft for my response.
💡 三个典型使用场景
1
📌 在 Outlook中使用：选取邮件，点击 Copilot 图标，直接贴入这个 Prompt 即可运行。
2
✍️ 草稿是起点，不是终点：记得根据场景调整语气与细节，加入你的个人风格。
3
🔍 追问更多信息：说「哪封信对我影响最大？」让 Copilot 帮你排出回应优先级。
