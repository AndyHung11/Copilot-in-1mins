忙完一整个会计年度，却说不出自己做了什么？这个 Prompt 让 Copilot 像 Spotify Wrapped 翻遍你 FY26 的日历、聊天、信件、文件，整理出大数字、口头禅、最佳拍档与最有戏的主旨，输出成一页可分享的精美 HTML。用一页回顾，好好替自己这一年庆祝一下 🎉
🎁 你将获得什么？
📊 你的大数字
主办了几场会、亮点时刻，4–5 个有凭有据的数字一次看清。
🎤 口头禅 + 最佳拍档
逐字抓出你最常说的话，和横跨整年的常态合作者。
🎨 一页精美 HTML
深色背景、霓虹配色、超大数字，自包含一页，直接分享。
🚀 直接复制这段 Prompt（简中版）
🎉 FY26 Wrapped（在 Copilot Chat / Cowork 打开 WorkIQ 后粘贴，把 [名字] 换成你的）

[名字] 的 FY26 Wrapped — 2025/7/1 到 2026/6/30，只使用这段期间的数据。

纳入任何内容前：每个模式需要 5 个以上的例子并附逐字引用。统计只算这个人亲自撰写或主办的。只有当某人出现在 4 季中的至少 3 季时才纳入排名。

日历：完整翻完四季（7–9 月、10–12 月、1–3 月、4–6 月）。计算主办的会议数与值得庆祝的亮点时刻。不要纳入个人医疗或家庭行程。

聊天：口头禅需 5 次以上确认出现、每次逐字引用。常用开场白附 5 个例子。

最佳拍档（Ensemble Cast）：依日历重叠、信件往来、聊天与共同工作，找出 FY26 的常态合作者；只纳入出现在 4 季中至少 3 季的人，并附佐证。

信件主旨：从寄件备份中挑 4–5 个有梗或好笑的。

文件：在 OneDrive 搜索意外有诗意、好笑或诚实得可疑的文件名。只纳入这个人创建或拥有的文件。

请给我：4–5 个大数字亮点、3–4 个口头禅、附佐证的前 5 名人物、2–3 句同事的逐字引言、一句收尾金句 🎤

输出成单一自包含的 HTML 档。所有 CSS 内嵌。背景 #0D0D1A。最多 4 个区块。字体用 Segoe UI Black 或 system-ui bold。数字大小 clamp(64px, 10vw, 120px)。强调色：#7C6AF7 · #F5A623 · #3DD68C。要有留白。Emoji 要对味——😭 混乱、💅 高光、💀 口头禅。语气：Beyoncé 巡演手册 ✕ 失控便利贴。
📋 Prompt Template (English)
🎉 FY26 Wrapped (turn on WorkIQ in Copilot Chat / Cowork, then paste and swap [Name])

[Name]'s FY26 Wrapped — Jul 1 2025 to Jun 30 2026. Only use data from this period.

Before including anything: patterns need 5+ examples with verbatim quotes. Stats must be things this person authored or organised. Only rank someone if they appear in at least 3 of the 4 quarters.

Calendar: paginate all four quarters fully (Jul–Sep, Oct–Dec, Jan–Mar, Apr–Jun). Count events organised and standout moments worth celebrating. No personal medical or family appointments obvi.

Chats: crutch words 5+ confirmed occurrences, verbatim each time. Go-to opener with 5 examples.

Ensemble Cast: recurring collaborators across FY26 based on calendar overlap, email activity, chats, and shared work; only include people who appear in at least 3 of the 4 quarters and provide supporting evidence.

Subject lines: 4–5 punchy or funny ones from sent items only.

Files: search OneDrive for doc names that are accidentally poetic, funny, or suspiciously honest. Only include files they created or own.

Give me: 4–5 big-number callouts, 3–4 crutch words, top 5 people with evidence, 2–3 verbatim colleague quotes, one closing line 🎤

Single self-contained HTML file. All CSS inline. #0D0D1A background. Max 4 sections. Segoe UI Black or system-ui bold. Numbers: clamp(64px, 10vw, 120px). Accents: #7C6AF7 · #F5A623 · #3DD68C. Breathing room. Emojis match the vibe — 😭 chaos 💅 wins 💀 crutch words. Tone: Beyoncé tour programme meets unhinged sticky note.
💡 三个典型使用场景
1
🔓 先打开 WorkIQ
这个 Prompt 要读你的日历、聊天、信件、OneDrive 等组织数据，务必在 Copilot Chat 或 Cowork 打开 WorkIQ（前称 Work 模式），否则只会得到空泛内容。
2
📅 提醒它分季翻完
数据量大时 Copilot 可能只抓近期，追问「四季都要完整翻过、每季各给佐证」确保整年涵盖。
3
🎨 嫌不够华丽就再要一次
输出 HTML 后可追问「数字再放大、加一点动画感、emoji 再到位」，或请它直接在 Cowork 里开成可分享页面。
