Copilot 的功能很多，但大多数人面对一大堆教学资源，往往不知道该从哪里下手，最后就是一直说「等我有空再学」——然后永远没有空。

这个 Prompt 让 Copilot 直接帮你设计一份个人化的学习计划——列出每天可以做的具体小任务，逐步把 Copilot 融入你的日常工作流程，同时每周推荐帮你快速入门的学习资源。

不需要一次学完所有功能，每天进步一点点，就是最好的导入策略！
🎁 你将获得什么？
📅 个人化的每日任务
小而可行的行动，融入你既有的工作流程。
📚 每周推荐资源
每周依你的职位和技能程度精选的教学。
🎯 工作流程集成技巧
把 Copilot 接进你既有工具的具体做法。
🚀 直接复制这段 Prompt（简中版）
请创建一份训练计划，帮助我导入 M365 Copilot。包含我每天可以进行的日常活动清单，让 Copilot 融入我的工作流程。每周加入实用资源，帮助我快速上手。
📋 Prompt Template (English)
Create a training plan to help me adopt M365 Copilot. Include a list of daily activities I can do to build Copilot into my routine. Add helpful resources to get me started each week.
💡 三个典型使用场景
1
🎯 指定你的职务
加上「我是 [职称]，主要工作是 [描述]」，让计划更贴近你的实际场景。
2
📌 从第一周任务开始
不用一次看完整份计划，先做第一周的任务，体验成就感再继续。
3
👥 拉同事一起学
把这个 Prompt 分享给你的团队，一起创建 Copilot 社区，互相分享心得。
