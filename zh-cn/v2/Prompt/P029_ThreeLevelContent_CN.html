同一份报告，高层要看摘要，中阶负责人要看细节，技术专家要看深度分析 — 三份要分开重写吗？不用！这个 Prompt 让 Copilot 把同一份内容自动分拆成三个版本：负责人摘要、中阶说明和深度分析，每个版本都针对目标受众量身订制。一份数据，三种版本，三组人都满意 🎯
🎁 你将获得什么？
👔 负责人摘要版
高度浓缩，只留关键结论和决策依据，一眼掌握重点。
📊 中阶说明版
补充背景和脉络，让负责人和组长了解全貌而不迷失在细节中。
🔬 深度分析版
完整细节、数据和技术说明，让专家和分析师得到所需深度。
⚡ 一次搞定三版
不需要重复作业，一个指令自动输出三个不同深度的版本。
🚀 直接复制这段 Prompt（简中版）
请将这份内容输出三个复杂度不同的版本：负责人摘要版、中阶说明版，以及深度分析版。
📋 Prompt Template (English)
Create three versions of this content with different complexity levels: executive summary, mid-level explanation, and in-depth analysis.
💡 三个典型使用场景
1
用 Copilot Chat 上传文档并运行这个 Prompt，三个版本依序输出。
2
输出后可以继续要求：「请将负责人摘要版控制在 100 字以内。」进一步精炼。
3
三个版本输出后，拷贝到不同的投视频或文档，方便依受众分别使用。
4
如果有特定受众，可以加上更多背景：「负责人摘要版要给非技术背景的 C-level 高管看。」
