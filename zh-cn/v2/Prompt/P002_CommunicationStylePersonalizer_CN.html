🚀 直接复制这段 Prompt（简中版）
请分析我过去 3 个月在 Outlook 与 Teams 的沟通内容（消息、回复、会议笔记），归纳出我的沟通风格，包含：
1. 语气（正式/口语、直接/委婉、严谨/轻松）
2. 结构偏好（条列、段落、开门见山或铺陈）
3. 常用词汇与避免词汇
4. 对不同对象（负责人、同事、客户）的语气差异
5. 我的内核沟通价值观

最后请输出两段个人化指令：
• 给 Copilot Chat 的 system prompt：未来请以这个风格回应我。
• 给 Outlook Draft with Copilot 的指令：未来起草信件时请套用此风格。
📋 Prompt Template (English)
I want you to help me create a set of instructions for the personalization feature in Copilot. Read through my recent emails and chats from Teams and provide a comprehensive analysis of my communication style by identifying my core values, strengths, weaknesses and skills. Based on the analysis, generate instructions that I can use in Copilot Chat and a second set for Outlook so that I can use it for e-mail drafting
