在 Outlook 中运行这个 Prompt，Copilot 自动摘要邮件串活动清单，独立条列所有行动项目，并以浅显易懂的语言整理，可直接转发给同事 📤
🎁 你将获得什么？
📋 详细活动清单
完整摘要邮件串中的所有事件与讨论，时序清楚，不遗漏任何重要细节。
✅ 独立行动项目清单
清楚条列出所有待办与行动项目，可直接转发给相关人员追踪。
🗣️ 浅显易懂的语言
用简单直接的语言整理，让同事马上能够接手，无需阅读原始邮件串。
📤 即可转发分享
在 Outlook 中直接生成，完成后可立即转发，节省沟通成本。
🚀 直接复制这段 Prompt（简中版）
请摘要这封邮件串的完整活动内容并以清单列出，然后另外产生一份行动项目的独立条列清单，可直接发送给我的同事。请使用浅显易懂的语言，让同事能快速进入状况。
📋 Prompt Template (English)
Create a summary of this email thread with detailed activities in a list and then create a separate Bullet list of actions items to be sent to my colleagues. Please use language that is easily to understand so my colleagues can get us up to speed quickly.
