每个支持团队都有这样的痛点：同样的客户问题一再出现，但每次解决的方式却不一样，质量不稳定，而且所有的「解法」都只存在老鸟的脑袋里……

这个 Prompt 让 Copilot 从你与 [人员/团队] 的讨论记录中，直接生成一份最佳实践文档——包含系统化的处理流程、过去成功案例的具体范例，以及应对常见客户问题的技巧清单。

让知识从个人记忆变成团队资产！
🎁 你将获得什么？
📋 一步步的处理流程
一套有系统的流程，处理每一种常见的客户问题。
📖 真实案例库
具体的过往案例，你的团队下次能直接拷贝套用。
✅ 常见抱怨技巧清单
针对每周都会出现的问题，给你速查的应对技巧。
🚀 直接复制这段 Prompt（简中版）
根据与 [人员／团队] 讨论 [主题] 相关客户问题的沟通记录，请创建一份更新版的最佳实践文档，概述解决类似问题的系统化方法。包含来自过去成功解决案例的范例，以及处理常见客户问题的技巧。
📋 Prompt Template (English)
Based on the discussion notes with [person / team] regarding the recent customer issue with [topic], create an updated best practices document that outlines a systematic approach for resolving similar complaints. Include examples from previous successful resolutions and tips for handling common customer questions.
💡 三个典型使用场景
1
📎 粘贴讨论记录
把与相关人员的邮件串、Teams 对话或会议摘要一起贴进来，让文档有真实依据。
2
📂 存进 SharePoint
生成后存入团队共享文档库，让所有成员都能随时查阅。
3
🔄 定期更新
每季回顾一次，请 Copilot 根据新的案例更新文档，保持内容的时效性。
