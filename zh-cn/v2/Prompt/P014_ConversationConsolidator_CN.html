你有没有花了好几轮对话，终于让 Copilot 给出完美的回答——但你根本不知道怎么重现它？下次想做类似的事，又要从头慢慢试，真的很浪费时间。

这个 Prompt 让 Copilot Chat 把整段对话浓缩成一个单一的、可重复使用的提示词，让你下次直接粘贴就能拿到相同质量的输出。特别适合用来创建你自己的提示词库，或自动化那些每周都要重复的例行性任务。
🎁 你将获得什么？
📋 整段对话完整摘要
把多轮对话的脉络和调整过程浓缩成清晰的摘要。
⚡ 单一可重用提示词
一个提示词就能重现最佳结果，不用再反复调整。
📚 创建个人提示词库
把精心调整出来的提示词收藏起来，随时取用。
🤖 自动化重复任务
适合每周都要跑的例行性任务，一次设置，反复使用。
🚀 直接复制这段 Prompt（简中版）
摘要这个对话串中的完整对话，并生成一个单一的、完整的提示词，可以重现我收到的最新回复。目标是将所有之前的提示和场景集成成一个可重复使用的输入。
📋 Prompt Template (English)
Summarize the entire conversation so far in this thread and generate a single, comprehensive prompt that would reproduce the latest response I received. The goal is to consolidate all previous prompts and context into one reusable input.
💡 三个典型使用场景
1
💾 立刻保存结果：跑完后把生成的提示词拷贝存到笔记或文档，创建你的个人提示词库。
2
🔧 微调后再存：收到集成提示词后，可以稍作调整让它更通用，适用于更多场景。
3
👥 分享给团队：把好用的提示词分享给同事，让整个团队都受益于你调整出来的最佳方案。
