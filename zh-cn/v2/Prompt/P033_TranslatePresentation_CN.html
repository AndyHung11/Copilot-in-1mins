🎁 你将获得什么？
🌐 完整翻译新演示文稿
自动创建全新翻译版，所有投视频文本翻译完整，格式与原版一致。
📋 原版完整保留
原始演示文稿不受影响，新版本另外保存，方便并行管理。
🗣️ 任意目标语言
支持翻译成任何你指定的语言，轻松应对不同市场需求。
⚡ 大幅节省时间
原本需要数小时的工作，几秒钟内完成，让你专注在内容质量上。
🚀 直接复制这段 Prompt（简中版）
请创建这份演示文稿的新版本，并将所有投视频翻译成 [选择语言]。
📋 Prompt Template (English)
Please create a new version of this slide deck and translate all the slides to [Choose Language]
💡 三个典型使用场景
1
把 [选择语言] 换成你需要的语言，例如：「日文」、「英文」、「西班牙文」、「韩文」等。
2
翻译完成后，快速浏览一遍，确认专业术语和产品名称是否翻译正确（建议保留英文原文）。
3
如果有特定用词要求，可以加上：「请保留以下词汇的英文原文：___」。
4
翻译版本适合作为初稿，建议有母语人士快速核阅一遍再正式使用。
