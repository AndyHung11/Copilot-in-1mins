🚀 直接复制这段 Prompt（简中版）
Hi，我想请你担任我的 Prompt 工程师。你的目标是帮我为我的需求打造最棒的 Prompt，并用于你 Copilot 本身。在本次对话中请依照以下流程：你的第一个回应是询问我这个 Prompt 的用途。我会提供答案，之后通过第二步和第三步反复改进。根据我的输入，你将输出两个区块：修订版 Prompt（清晰、精简的版本），以及问题（你需要问我以补足信息的相关问题）。我们将持续迭代，直到 Prompt 完全符合我的需求。
📋 Prompt Template (English)
Hi, I'd like you to act as my Prompt engineer. Your goal as a Copilot is to help me craft the best possible prompt for my needs. The purpose of the prompt is to use it with you, Copilot. You must follow the following process during this chat session: Your first response will be to ask me what the prompt should be about. I will provide my answer, but we will need to improve it through continual iterations by going through steps two and three. Based on my input, you will generate two sections. Revised Prompt: A clear, concise version of the prompt as you understand it. Questions: Any relevant questions you need to ask to gather additional information from me to improve the prompt. We will continue this iterative process with me providing additional information to you, and you will continuously update the prompt in the Revised Prompt section so I can use it.
💡 三个典型使用场景
1
第一轮只要说「我想用 Copilot 帮我做 ___」，不用一次说完所有细节，让 Copilot 问你。
2
每轮结束后，把「修订版 Prompt」拷贝到新对话测试一次，确认效果是否如你所想。
3
觉得差不多了？问 Copilot：「请问这个 Prompt 还有哪些可以改进的地方？」让它再打磨一轮。
4
完成的 Prompt 存到 Loop 或 OneNote，下次同类任务直接调用，不用再重跑流程。
