参加了一场超级精彩的会议，信息量大到有点消化不过来？大家在聊天室里问了很多问题、分享了很多数据，但会议一结束，那些宝贵的讨论就消失在记录的洪流中。

这个 Prompt 让 Copilot Chat - Work 把你的会议聊天记录转化成一份结构化的问答文档，使用 Markdown 格式清晰分层，关键信息加粗标示，让整个团队都能快速查阅最重要的讨论内容。
🎁 你将获得什么？
📋 完整问答文档
从聊天记录萃取最重要的问题与解答，整理成可查阅的文档。
📊 Markdown 格式清晰呈现
使用清楚的标题、条列和粗体，让文档结构一目了然。
🔍 关键信息高亮标示
重要信息自动以粗体标示，快速找到你需要的内容。
📁 便于查阅与分享
文档格式清晰，方便在团队中分享或存入知识库。
🚀 直接复制这段 Prompt（简中版）
根据 [会议聊天记录] 的会议聊天记录，生成一份包含最重要问题的完整问答文档。使用 Markdown 格式设计文档，包含每个问题和答案的清晰标题、重点的条列格式，以及粗体标示重要信息。确保文档视觉清晰且易于导览。
📋 Prompt Template (English)
Based on the meeting chat of [meeting chat], generate a comprehensive Q&A document with the most interesting questions. Use Markdown to format the document, including clear headings for each question and answer, bullet points for key points, and bold text to highlight important information. Ensure the document is visually appealing and easy to navigate.
💡 三个典型使用场景
1
📋 提供聊天记录：把 Teams 聊天记录拷贝或上传，替换 [会议聊天记录] 后运行。（若改让 Copilot 直接读取 Teams 会议则需高端许可）
2
🎯 指定关注的问题类型：加上「特别关注技术问题和待办任务」让文档更聚焦。
3
📤 分享给未出席者：问答文档是让没有出席的同事快速跟上的最佳工具。
