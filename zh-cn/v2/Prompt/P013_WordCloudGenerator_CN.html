识别文档的内核主题
你有没有看过那种很酷的文本云可视化，大大小小的字飘在一起，一眼就看出哪些主题最重要？

这个 Prompt 让你把任何文档上传给 Copilot Chat - Web，只要一句话就能生成你自己的文本云。无论是会议记录、研究报告、客户反馈还是年度报告，几秒钟后就能看到内核主题一览无遗。不需要任何设计工具，也不需要 M365 许可。
🎁 你将获得什么？
☁️ 文本云可视化图片
直观呈现文档最重要的关键字，可直接加入演示文稿或报告。
🔤 前10大关键字提取
自动识别并列出最具代表性的词汇。
🚫 过滤填充词
自动去除无意义的常用词，聚焦真正有意义的主题词汇。
📁 多格式支持
支持 Word、PowerPoint、Excel、PDF 及纯文本文件。
🚀 直接复制这段 Prompt（简中版）
创建 [text-document.docx] 中前10大关键字的文本云。请忽略填充词。
📋 Prompt Template (English)
Create a Word Cloud of the top 10 key words in [text-document.docx]. Ignore filler words.
💡 三个典型使用场景
1
📎 上传文档再运行：把 Word、PDF、PowerPoint 或 Excel 文件上传给 Copilot Chat（Web） 后再粘贴这个 Prompt。
2
🔢 调整关键字数量：把「前10大」改成「前20大」可以获得更丰富的文本云。
3
🌐 加入语言指定：若文档是英文，追问「请也显示中文翻译」让结果更易读。
